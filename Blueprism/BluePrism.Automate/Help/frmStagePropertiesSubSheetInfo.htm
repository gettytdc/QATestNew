Stage Types

Page Information Stage

The page properties dialog allows you to enter a description of the page.

1. Enter a name

The name that you enter identifies the information, it will be displayed on the process diagram

2. Enter a description

The additional description field allows a lengthier narrative to be entered.

3. Set Pre/Post-conditions

The pre/post-conditions boxes are a place where you may informally define criteria that must be met before the process runs (preconditions) and goals that the page must achieve by the end of the page (postconditions). Blue Prism will not perform any sort of checking on these conditions; they exist merely to help anyone developing or maintaining a process to keep track of how a complex process is intended to work.
