System Manager - Update License

Introduction

To use Blue Prism, a valid license must be installed. The license is installed into the database via System Manager, and applies to the whole Blue Prism environment associated with that database.

Installing a license

After receiving a license key from Blue Prism, it can be installed as follows :-

  1. Log into an instance of Blue Prism
  2. Navigate to System Manager
  3. Select the "License" page in the navigation menu.
  4. Click "Install a new license" and select the license file

License Assistance

For any further advice or assistance regarding licenses, please contact Blue Prism.
