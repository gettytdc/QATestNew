アプリケーションの属性 - HTMLモード

はじめに

以下は、HTMLモードのWindowsスパイツールから返される属性のリストです。他のモードから返される属性のリストについては、以下を参照してください。

  • Win32モード
  • アクティブアクセシビリティモード
  • UIオートメーションモード
  • Javaモード

アプリケーションの属性

X
その親要素を基準とした要素の左上隅の水平座標です。Y、幅、高さも参照してください。 各座標単位は1画面のピクセルを表します。
Y
その親要素を基準とした要素の左上隅の垂直座標です。X、幅、高さも参照してください。 各座標単位は1画面のピクセルを表します。
幅
要素の幅（画面のピクセル単位）です。 X、Y、高さも参照してください。
高さ
要素の高さ（画面のピクセル単位）です。 X、Y、幅も参照してください。
パス
ドキュメントオブジェクトモデル（DOM）内の要素のパスのテキスト表現です。たとえば、パスHTML/BODY(1)/TABLE(4)は、DOMのhtml要素の下の最初のbody要素の下で宣言されている、4番目のtable要素を参照しています。
TagName
HTMLドキュメントソースで宣言されている要素を表すタグの名前です。たとえば、ハイパーリンクのタグ名は「a」です。ハイパーリンクは、htmlの文字aで表されるアンカータグから作成されます。
ClassName
カスケードスタイルシート（CSS）で使用されている、要素を表すタグのクラス名です。たとえば、HTMLタグ<div class="navigation">では、クラス名は「navigation」です。
ID
htmlソースで指定されている、要素を表すhtmlタグのIDです。 たとえば、タグ<a id="Section3>のIDは「Section3」です。
タイトル
要素を表すHTMLタグのタイトルです。
リンク
たとえば、ハイパーリンクのターゲットURLです。
InputType
入力フォーム内の要素にのみ関係します。この属性は入力するデータの種類です。
InputIdentifier
入力フォーム内の要素にのみ関係します。この属性は、対象の入力要素の一意の識別子です。
値
要素に表示される値です（ある場合）。編集フィールド、チェックボックス、ラベルなどの要素にのみ関係します。
有効
要素が有効になっているかどうかを示します。有効になっていない要素は、キーボードまたはマウスの入力を受け取ることができません。
チェック済み
要素がチェックされているかどうかを示します。入力フォームのチェックボックスなど、True/False、はい/いいえ、オン/オフの値を表す要素にのみ関係します。
