ヘルプ

バージョン

新しいバージョンのBlue Prismにはより多くの機能があります。ご使用のバージョンのBlue Prismにはない機能についてご興味をお持ちの場合は、お問い合わせください。

以下はBlue Prismの各バージョンに追加された主な機能の内訳です。

バージョン5.0.x

  • インサイトと分析
    • 管理ダッシュボード
    • 傾向分析
    • 使用率と完了率
    • 依存関係の追跡
  • デジタルワークフォース
    • アクティブキュー
    • コントロールルームフィルター
    • ロギングレベルの追加
  • 拡張性
    • 多層グループ化構造
    • 認証情報マネージャーの拡張
    • レイテンシの高い環境に対するサポートの向上
    • ロケールと国際文字セットのサポート
  • 開発と実装
    • グローバルアプリケーションモデル
    • アクションまたはページのコピーと貼り付け
    • 結合文字のフォント認識
    • リリースマネージャーの更新
    • OCR機能
    • バージョン履歴の卓越性（プロセスとオブジェクト）
  • ユーザーエクスペリエンス
    • ユーザーインターフェイスの更新
    • プロセスの組み合わせとオブジェクトスタジオ
    • 環境テーマ
  • エンタープライズプラットフォーム
    • .NET 4プラットフォームのサポート
    • SQL Azureのサポート
    • SQL AlwaysOn可用性グループのサポート
    • セキュリティ保護可能なデータベースオブジェクトをコントロールするパブリッシュ済みSQLの役割

バージョン4.2.x

  • 64ビットのサポート
  • より優れたユーザーアクセスコントロールのサポート
  • 強化された文字マッチング/領域エディター
  • カレンダー内部ビジネスオブジェクト
  • アプリケーションモデルのフィルタリング
  • 強化されたセッションログ検索/フィルタリング
  • 暗号化ワークキュー
  • 暗号化ビジネスオブジェクト
  • スケジュールされたタスクをセッション失敗後も継続
  • 強化されたコードステージエディター
  • テキストデータアイテムにおける複数行のテキストのサポート
  • 新しいデータ型変換関数
  • より強力なパスワードポリシールール

バージョン4.1.x

  • リリースマネージャー
  • デザインコントロール - 高度な検証
  • フォント認識
  • メモリ診断
  • 環境定数
  • セッション変数 - コントロールルームから編集可能
  • 製品に含まれているユーティリティVBO

バージョン4.0.x

  • 入れ子のコレクション
  • 複合オブジェクト（単一行コレクションとして実装）
  • Webサービスサポートの強化

バージョン3.5.x

  • ワークキュー管理
  • 例外処理
  • 認証情報マネージャー
  • リソースプール
  • プロセスのグループ化
  • スケジューラー
  • Blue Prismサーバー

バージョン3.0.x

  • オブジェクトスタジオ
  • アプリケーションモデラー（別名インテグレーションアシスタント）
  • プロセスMI

バージョン2.1.x

  • ログファイルのエクスポート
  • デバッグの改善（可変速度、ステップアウト）
  • コレクションの作成
  • パブリッシュ/廃止プロセス
  • プロセスの違い
  • データアイテムウォッチ

バージョン2.0.x

  • プロセス監査
  • プロセスバックアップ
  • プロセス検索
  • 強化されたステージプロパティGUI
  • コントロールルームのパフォーマンスの向上

バージョン1.1.x

  • ビジネスオブジェクトの構成
  • プロセススタジオのページ
