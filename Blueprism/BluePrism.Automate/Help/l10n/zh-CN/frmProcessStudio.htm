Studio

概述

Studio 是 Blue Prism 的设计区域，在这里创建、修改流程和对象，并进行初始测试，然后将其发布到控制室以供使用。

Process Studio 使用起来很简单：通过使用部署在当前计算机上的资源，拖放界面可以快速、轻松地实现流程流。

Studio 选项卡显示在树和列表视图控件中成组的现有流程和对象。在树中选择一个组后，系统将显示在右侧列表中该组的成员，而在树中选择一个流程（或对象）后，系统将显示其版本历史记录。右键上下文菜单提供了多个选项。

组上的挂锁图标表示该组权限受限。通过上下文菜单选择“管理权限”选项，可以查看该权限。要在组上设置权限，用户需要被分配一个具备“管理权限”的权限的角色

创建新流程

从头开始创建新业务流程的分步指南。在迁移至 Process Studio 开始工作之前，您将了解确定新流程所需的初始步骤。

编辑现有流程

系统会显示现有流程供您选择。选择流程后，可在 Process Studio 中编辑、测试和保存流程。

克隆现有流程

此方法可基于现有流程或模板快速轻松地创建新流程―克隆选项将获取已保存的流程，请求新名称，复制该流程，然后将其移动到 Process Studio 中并加载该流程。

删除流程

从数据库中删除现有流程。
