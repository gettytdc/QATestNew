应用程序属性—Active Accessibility 模式

简介

以下是 Windows 探测工具在 Active Accessibility 模式中返回的属性列表。有关由其他模式返回的属性列表，请参阅：—

  • Win32 模式
  • UI Automation 模式
  • Java 模式
  • HTML 模式

应用程序属性

已选中
仅与某些类型的元素相关，如指示 True/False 或 On/Off 状态的复选框、切换按钮、单选组等。此属性指示是否选择了某个选项。另请参阅混合属性。
已折叠
仅与某些类型的元素相关，如具有子元素的组合框、树视图节点等。 此属性指示选项是否已折叠（折叠意味着其子项是隐藏的）。
已展开
仅与某些类型的元素相关，如具有子元素的组合框、树视图节点等。 此属性与已折叠属性相反。
可聚焦
指示元素是否可聚焦；聚焦的元素接收任何键盘输入。另请参阅已聚焦属性。
已聚焦
指示元素是否已聚焦（聚焦意味着它接收任何键盘输入）。这是一种瞬时状态，会根据用户的行为随时间变化。另请参阅可聚焦属性。
不可见
指示元素是否在屏幕上可见。位于屏幕外的任何元素都将不可见。
已链接
当元素的类型是超链接时为 True。
混合
仅与某些类型的元素相关，如指示 True/False 或 On/Off 状态的复选框、切换按钮、单选组等。如果切换状态不确定，则状态为“混合”。例如，复选框可以在既未设置 (True) 也未不设置 (False) 的状态下开始，这表示用户尚未选择选项。在这种情况下，状态被认为是不确定的。 另请参阅已选中属性。
可移动
指示是否可以移动元素。
可多选
指示是否可以同时选择元素中的多个项目。仅与允许选择项目的列表视图等元素相关。
已保护
指示是否使元素中包含的数据不可见，如在密码字段中。
只读
指示是否可以更改元素中的数据。
可选择
指示是否可以选择元素。另请参阅选定属性。
选定
指示是否已选择元素。另请参阅可选择属性。
可调大小
指示元素是否可以调整大小。
不可用
指示元素是否可用于交互。 对应于 Win32 应用程序的已启用属性。
