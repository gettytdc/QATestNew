产品说明

兼容性指南

概述

Blue Prism 平台可以在多种架构上运行，并可与众多产品和系统实现无缝对接。但是，在某些配置中运行这个软件时，会出现一些兼容性问题。

下方列举了可能会出现的问题，但是所列问题并非详尽无遗。

操作系统

Windows 2000
Active Accessibility 在 Windows 2000 上并不可靠，尤其是在与 VB6 应用程序对接时。
64 位 Windows
Blue Prism 服务器在 64 位 Windows 上受支持；但是，Blue Prism 客户端目前无法在 64 位环境中可靠地运行。
Windows XP；32 位 Windows Vista；32 位 Windows 7
没有已知的兼容性问题

数据库

SQL Server 2000
某些工作队列优化不适用于 SQL Server 2000，这意味着工作队列图形界面在处理非常大型的队列时运行速度可能会有点慢。
SQL Server 2005、SQL Server 2008
没有已知的兼容性问题
