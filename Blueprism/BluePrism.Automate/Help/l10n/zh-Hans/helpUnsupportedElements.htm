帮助

以下是给定应用程序类型不支持的元素列表。这意味着尽管该元素可探测，但是读写和导航操作对这些元素不起作用。

基于 Visual Studio 2005 的应用程序

  • 调节按钮―使用 Active Accessibility

基于 Visual Basic 6.0 的应用程序

  • 标签―使用 Active Accessibility
  • 调节按钮―使用 Active Accessibility

基于 Web 的应用程序

  • 标签―使用 Active Accessibility
  • 列表框―使用 Active Accessibility
  • 组合框―使用 Active Accessibility

基于 Java Swing 的应用程序

  • 滚动条―不支持写入
  • 轨迹条―不支持写入

基于 Java AWT 的应用程序

  • 列表框―不支持
  • 滚动条―不支持
  • 菜单―不支持
  • 菜单项―不支持
