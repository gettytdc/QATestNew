Blue Prism

当前队列总量

BPDS_QueueVolumesNow

返回执行时指定工作队列的项数（按状态列出）。

来源：Blue Prism 数据库—实时表

示例图表

参数

参数名称 数据类型 描述 验证
@BPQueueName 文本 要查询的单个队列的名称（如未指定，将查询所有队列） 无。
@ExcludePending 文本 设置为“True”可从图表中排除挂起的项（如未指定，挂起的项将包含在内） True 或 False
@ExcludeDeferred 文本 设置为“True”可从图表中排除延迟的项（如未指定，延迟的项将包含在内） True 或 False
@ExcludeComplete 文本 设置为“True”可从图表中排除完成的项（如未指定，完成的项将包含在内） True 或 False
@ExcludeExceptions 文本 设置为“True”可从图表中排除异常（如未指定，异常将包含在内） True 或 False
