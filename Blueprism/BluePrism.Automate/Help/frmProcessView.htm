View an Existing Process

A quick and easy way to view an existing process is by using the View Process wizard, from the Process Studio main menu. Follow the steps below as indicated by the numbered options on screen.

1. Locate a process

A list of existing processes from the database is presented with name, version and description details listed. The information in the box can be sorted according to each column by clicking the appropriate heading.

To select a process to view, simply highlight the entry required using the mouse.

2. Continue to Process Studio

Once you have completed step 1 clicking OK will open the selected process and progress to Process Studio with your process loaded onto the canvas. You will be able to view all aspects of the process but will not be able to modify it in any meaningful way.
