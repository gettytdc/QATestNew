Stage Types

End Stage

The end stage properties dialog has a number of fields to allow you to define what happens at the end of the process.

1. Enter a name

The name that you enter identifies the end stage, it will be displayed on the process diagram.

2. Enter a description

The additional description field allows a lengthier narrative to be entered.

3. Enter outputs

Outputs of the process are specified here. These relate to parameters that are passed out of the process when the process is run. To add a parameter simply click the grid below the name column to add a name to the parameter. choose a datatype below the data type column. Finally select a data item to store the input parameter in by dragging a data item from the data explorer and dropping it into the value column.
