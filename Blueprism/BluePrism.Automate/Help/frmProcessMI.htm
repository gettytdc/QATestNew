Process Management Information

The Process MI form is used to analyse previous sessions of a process. Session log data is extracted from the database and overlaid on the process diagram. Here, "MI" stands for "Management Information."

The use of this feature is subject to licensing restrictions, and as such it may be disabled in the user interface. If you are interested in using this feature, please contact Blue Prism to purchase a new license.

Using the Process MI Form

The top half of the form is used to search for sessions. Select one or more Resources, enter a Start Date and an End Date and then press the Search button. Blue Prism will then display a list of sessions that were run on those Resources between those dates

The bottom half of the form is used to select the parts of the process to analyse. A list of Action, Calculation and Decision stages is displayed on the left. Tick the parts of the process to be included in the analysis and press the Analyse button. Blue Prism will then extract session data for the selected stages and display it on the process diagram. Once complete the analysis can be saved in a text file.

Note that the analysis may take time depending on the quantity of session data in the Blue Prism database. You may reduce the time necessary by reducing the number of sessions and stages for which you desire an analysis.
