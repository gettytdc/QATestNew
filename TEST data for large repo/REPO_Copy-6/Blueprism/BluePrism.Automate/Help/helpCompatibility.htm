Product Notes

Compatibility Guide

Overview

The Blue Prism platform operates on a number of architectures, interfacing seamlessly with numerous products and systems. There are, however, some compatibility issues associated with running the software in certain configurations.

A non-exhaustive list of potential issues is enumerated below.

Operating Systems

Windows 2000
Active Accessibility is unreliable on Windows 2000 - especially when interfacing with VB6 applications.
64-bit Windows
Blue Prism Server is supported on 64-bit Windows, but currently the Blue Prism client can not reliably operate in a 64-bit environment.
Windows XP; 32-bit Windows Vista; 32-bit Windows 7
No known compatibility issues

Databases

SQL Server 2000
Some work queue optimisations are unavailable in SQL Server 2000, meaning that the work queues graphical interface may operate a little slowly with very large queues.
SQL Server 2005, SQL Server 2008
No known compatibility issues
