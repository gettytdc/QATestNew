Stage Types

Loop End Stage

For an overview of the uses of loops, see the loop start help page. The loop end stage is used to define at which point the loop will advance to the next item of the collection about which the loop is being performed. There is no functional information to be added to the loop end stage, but you may modify the stage's name and add a description.

1. Enter a name

The name that you enter identifies the end of the loop, it will be displayed on the process diagram.

2. Enter a description

The additional description field allows a lengthier narrative to be entered.
