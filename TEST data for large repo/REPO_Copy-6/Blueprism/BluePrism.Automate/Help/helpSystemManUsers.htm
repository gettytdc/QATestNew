System Manager - Manage Users

The User Management module is used to administer the users and the associated roles that grant them permissions within the Blue Prism platform.

The Security - Users tab displays the existing users organised into their groups in a tree control on the left hand side. Selecting a group in the tree shows the members of that group in a list on the right, whereas selecting a user in the tree shows the user details pane on the right.

The Users tab allows users to be organised or grouped into a hierarchical tree structure. The actions that can be taken are available via right-click context menus.

The User List View:

Displays a list of users and summary information about each user record.

The context menu displays the options available based on the authentication mode being used; and provides different options based on whether a user is currently selected.

The User Details View:

Displays read-only information about the user.

The context menu displays the options available based on the authentication mode being used; and provides different options based on whether a user is currently selected.

Context Menu Options

Grouping Users

Using the context menu on the user tree on the left hand side, users may be organised into groups.

Show All Users

The context menu on the user tree has an option to Show All Users, which toggles the tree and list view to display users who have been deleted from the system.

Create a new user

Create new Blue Prism users and assign roles which allow permissions within the Blue Prism platform.

Edit an existing user

Edit an existing user to change attributes, reset passwords or re-assign roles.

Delete an existing user

Delete a user who no longer needs access to the system. Deleted users do not show in the user list but can be made visible by right clicking the users tree and selecting show all users.

Only available when configured to use Blue Prism native authentication

Unlock a user

Unlock users who have locked themselves out by entering the incorrect password too many times, as configured in the log in options .

Only available when configured to use Blue Prism native authentication

Refresh User List

Only avaiable when configured to use Single Sign-on for Blue Prism - refresh the current user list based on Active Directory group membership.

View User's Group Membership

View the user's Active Directory group membership. Only available when configured to use Single Sign-on for Blue Prism..

Configure Process Alerts

Edit the configuration of process alerts for the selected user.
