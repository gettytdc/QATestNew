Block Stage

Blocks allow areas on the process diagram to be marked out. Stages falling completely inside these areas will all utilise the same recover stage when an exception occurs. For an overview of exceptions see: Exceptions Help

1. Enter a name

The name that you enter identifies the block, it will be displayed on the process diagram.

2. Enter a description

The additional description field allows a lengthier narrative to be entered.
