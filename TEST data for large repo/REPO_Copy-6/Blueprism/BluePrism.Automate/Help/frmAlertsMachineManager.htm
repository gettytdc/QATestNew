Process Alerts Machine Manager

Overview

Individual machines must register before they may receive process alerts. This allows Blue Prism to monitor the number of seats using this feature, for licensing purposes.

This registration takes place automatically when you use the process alerts configuration form; however you may need to be aware of it in case you need to manually remove a registered machine.

About this utility

The alerts machine management tool allows you to remove machines which have registered to use Process Alerts. This is likely to be useful when you are no longer using the machine for process alerts. Removing that machine allows for another machine to be registered instead (up to the number permitted by your Blue Prism License.

Using the form

The form can be accessed from System Manager.

Select all machines that you wish to delete; leave all others unselected. Click the delete button. The machines will be deleted instantly.

Note that this may have an adverse effect on users who are currently using process alerts on any of the machines in question. You should check that no-one is using process alerts on each machine before you proceed.
