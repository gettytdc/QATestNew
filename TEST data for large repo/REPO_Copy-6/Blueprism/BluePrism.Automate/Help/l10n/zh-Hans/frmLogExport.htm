导出日志

导出日志的第一步是在日志查看器中打开日志，然后从文件菜单中选择所需的导出选项。

接下来，您必须选择所需的文件格式。有三种文件格式可供选择：纯文本文件（如日志查看器本身所示）；html 文件（适用于 Web 浏览器中的标记）；以及 csv 文件（表示逗号分隔值，此文件格式用于格式化表格数据，使其适合在电子表格中使用）。如果您选择 html 或 csv，则日志的每一行中的数据将被分为消息日期时间、消息和 stageid 等不同列（如适用）。如果您不确定要选择哪种文件格式，请选择默认文本 (.txt)。

您还必须在第一步决定是否要包含环节 ID。日志查看器的帮助页面简要描述了环节 ID。如果不了解环节 ID 的用途，则您可能不希望将其包含在内。但是，如果打算将导出的日志发送到 Blue Prism 以获得技术支持，那么您应该包含环节 ID。

点击“下一步”后，系统将提示您为导出的日志输入文件名。您可以直接键入或浏览特定位置。如果您使用浏览功能，则系统将根据会话的 ID 及其运行日期提供一个建议的文件名。

点击“下一步”后，系统将通过进度条给出导出流程的进度。如果日志较大或数据库服务器负荷很重，您可能需要耐心等待。在等待导出以继续的同时，您可以返回日志查看器（或实际上是 Blue Prism 的任何其他部分）并继续工作。
