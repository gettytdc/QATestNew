流程导出

注意： 此机制基本上已被发行版管理器取代。有关在发行版中导出流程的具体详细信息，请参阅新建发行版向导的帮助文档。

您仍然可以使用导出流程功能将流程和可视业务对象导出到由本软件早期（4.1 之前）版本管理的 Blue Prism 环境中。

概述

流程导出功能允许您在当前 Blue Prism 数据库中创建流程（或可视业务对象）的本地副本。它将存储为 XML 文件格式，以便人工检查和通过电子邮件传输。

导出流程/业务对象的原因包括：

  • 将其从测试数据库复制到实时数据库（从测试数据库导出，然后再导入到目标数据库中）；
  • 将其发送给 Blue Prism 以寻求技术支持。

导出流程

要导出流程（或可视业务对象），请按照以下步骤操作：

  • 启动 Process Studio 后，从主下拉菜单中选择“文件”。
  • 选择“导出”功能以打开“流程导出”向导。
  • 根据需要选择“流程”或“业务对象”，然后点击“下一步”。
  • 从列表中选择相关项，然后点击“下一步”。
  • 输入文件位置以存放导出的流程。这可以是新文件名，或者，您也可以覆盖现有的文件。
  • 点击“完成”以完成操作。如果您已选择覆盖现有文件，系统将提示您进行确认；如果您改变主意，您将有机会选择不同的文件名。
