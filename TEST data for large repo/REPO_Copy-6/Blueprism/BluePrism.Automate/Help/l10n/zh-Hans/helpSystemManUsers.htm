系统管理器—管理用户

用户管理模块用于管理用户以及在 Blue Prism 平台内向他们授予权限的相关角色。

“安全—用户”选项卡左侧的树控件中显示组织到其组中的现有用户。在树中选择组会在右侧的列表中显示该组的成员，而在树中选择用户会在右侧显示用户详细信息面板。

使用“用户”选项卡，可以将用户组织或分组为分层树状结构。右键上下文菜单提供了可执行的操作。

用户列表视图：

显示有关每个用户记录的用户列表和摘要信息。

上下文菜单根据正在使用的身份验证模式显示可用选项；并根据当前是否选择了用户提供不同的选项。

用户详细信息视图：

显示有关用户的只读信息。

上下文菜单根据正在使用的身份验证模式显示可用选项；并根据当前是否选择了用户提供不同的选项。

上下文菜单选项

分组用户

使用左侧用户树上的上下文菜单，可以将用户组织到组中。

显示所有用户

用户树上的上下文菜单具有“显示所有用户”选项，可以切换树和列表视图以显示已从系统中删除的用户。

创建新用户

创建新的 Blue Prism 用户并分配允许在 Blue Prism 平台内执行权限的角色。

编辑现有用户

编辑现有用户以更改属性、重置密码或重新分配角色。

删除现有用户

删除不再需要访问系统的用户。已删除的用户不会显示在用户列表中，但可以通过右键点击用户树并选择“显示所有用户”来显示。

仅在配置为使用 Blue Prism 本机身份验证时可用

解锁用户

解锁由于多次输入错误密码而按照登录选项中的配置将自己锁定的用户。

仅在配置为使用 Blue Prism 本机身份验证时可用

刷新用户列表

仅在配置为对 Blue Prism 使用单点登录时可用—根据 Active Directory 组成员身份刷新当前用户列表。

查看用户的组成员身份

查看用户的 Active Directory 组成员身份。仅在配置为使用 Blue Prism 的单点登录时可用。

配置流程警报

编辑所选用户的流程警报配置。
