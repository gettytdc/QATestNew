控制

会话管理

概述

控制室是 Blue Prism 的区域，供您在实际环境中控制已设计、测试和部署的流程。

控制室中的“环境”显示流程、资源和会话。

流程定义为以前通过 Process Studio 设计和测试的业务流程。它通过执行预先开发的操作并根据其结果做出决策来封装特定的业务流程。

资源是执行全部或部分流程的单元，可以是人，也可以是更常见的计算机。

会话表示当前分配给资源的流程。它保持一种状态（例如挂起、正在运行、已完成），并且可以在视图中筛选以查看相关数据。

控制室显示当前环境中的所有可用资源和流程。

操作

创建会话

为了启动新会话，将流程拖到所需的资源上，或将资源拖到所需的流程上。

提示：要在多个资源上快速运行一个流程，通过拖动鼠标选择多个资源，并将整个选择拖到单个流程上。

挂起的会话将显示在窗口下半部分的列表视图中，并显示为橙色。

启动会话

要启动挂起的会话，选择所有所需的会话（使用 Ctrl 键选择多个会话），然后点击“开始选择”链接，或右键点击所选内容并选择“开始”。

停止会话

要“立即”停止正在运行的会话或一组会话，选择所有所需的会话（使用 Ctrl 键选择多个会话），然后点击“停止选择”链接，或右键点击所选内容并选择“立即停止”。

要请求“安全”停止正在运行的会话或一组会话，选择所有所需的会话（使用 Ctrl 键选择多个会话），然后右键点击所选内容并选择“请求停止”。系统将在会话上设置一个标志，正在运行的流程可以使用 IsStopRequested() 环境函数检查该标志，从而能干净地退出当前操作。
