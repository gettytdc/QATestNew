版本

较新版本的 Blue Prism 具有更多功能，如果您对您的 Blue Prism 版本中不存在的功能感兴趣，请联系我们。

以下是每个版本的 Blue Prism 中添加的主要功能的明细

版本 5.0.x

  • 洞见和分析
    • 管理仪表盘
    • 趋势分析
    • 利用率和完成率
    • 依赖关系跟踪
  • 数字化劳动力
    • 活动队列
    • 控制室筛选器
    • 更多日志记录级别
  • 可扩展性
    • 多层分组结构
    • 凭据管理器扩展
    • 更好地支持更高延迟的环境
    • 支持语言环境和国际字符集
  • 开发和实施
    • 全局应用程序模型
    • 复制并粘贴操作或页面
    • 连接字符的字体识别
    • 发行版管理器更新
    • OCR 功能
    • 版本历史记录突出显示（流程和对象）
  • 用户体验
    • 刷新了用户界面
    • 组合了 Process 和 Object Studio
    • 环境主题
  • 企业平台
    • .NET 4 平台支持
    • SQL Azure 支持
    • SQL AlwaysOn 可用性组支持
    • 发布了用于控制数据库安全性的 SQL 角色

版本 4.2.x

  • 64 位支持
  • 改善了用户访问控制支持
  • 改进了字符匹配/区域编辑器
  • 日历内部业务对象
  • 应用程序模型筛选
  • 改进了会话日志搜索/筛选
  • 加密了工作队列
  • 加密业务对象
  • 计划的任务在会话失败后继续
  • 改进了代码环节编辑器
  • 文本数据项中的多行文本支持
  • 新的数据类型转换函数
  • 更强大的密码策略规则

版本 4.1.x

  • 发行版管理器
  • 设计控制—高级验证
  • 字体识别
  • 内存诊断
  • 环境常数
  • 会话变量—可从控制室编辑
  • 在产品中包含实用程序 VBO

版本 4.0.x

  • 嵌套式集合
  • 复合对象（作为单行集合实现）
  • 增强了 Web 服务支持

版本 3.5.x

  • 工作队列管理
  • 异常处理
  • 凭据管理器
  • 资源池
  • 流程分组
  • 调度程序
  • Blue Prism 服务器

版本 3.0.x

  • Object Studio
  • 应用程序建模器（也称为集成助手）
  • 流程 MI

版本 2.1.x

  • 日志文件导出
  • 调试改进（可变速度、单步跳出）
  • 创建集合
  • 发布/停用流程
  • 流程区分
  • 数据项监视

版本 2.0.x

  • 流程审计
  • 流程备份
  • 流程搜索
  • 改进了环节属性 GUI
  • 控制室性能增强

版本 1.1.x

  • 业务对象配置
  • Process Studio 中的页面
