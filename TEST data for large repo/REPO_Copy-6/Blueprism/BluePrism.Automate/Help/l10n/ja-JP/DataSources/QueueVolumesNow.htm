BPDS_QueueVolumesNow

実行時に指定されたワークキューのステータス別のアイテム数を返します。

ソース：Blue Prismデータベース - ライブテーブル

サンプルチャート

パラメーター

パラメーター名 データ型 説明 検証
@BPQueueName テキスト 照会する個々のキューの名前（指定しない場合、すべてのキューが照会されます） なし。
@ExcludePending テキスト チャートから保留中のアイテムを除外するには「True」に設定します（指定しない場合、保留中のアイテムが含まれます） TrueまたはFalse
@ExcludeDeferred テキスト チャートから延期アイテムを除外するには「True」に設定します（指定しない場合、延期アイテムが含まれます） TrueまたはFalse
@ExcludeComplete テキスト チャートから完了アイテムを除外するには「True」に設定します（指定しない場合、完了アイテムが含まれます） TrueまたはFalse
@ExcludeExceptions テキスト チャートから例外を除外するには「True」に設定します（指定しない場合、例外が含まれます） TrueまたはFalse
