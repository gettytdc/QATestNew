印刷プレビュー

印刷プレビューでは、プロセスを印刷する前に印刷設定を確認できます。プリンターと用紙の設定を調整するだけでなく、用紙に合わせてプロセスを拡大縮小、移動、回転することもできます。印刷プレビューに表示されるボタンとドロップダウンは次のとおりです。

印刷
印刷設定ダイアログボックスを表示します
ページ設定
［ページ設定］ダイアログボックスを表示します
レイアウトドロップダウン
プレビューに使用するページレイアウトを選択できます
全体表示
現在のページレイアウトに合わせてプロセス図を拡大縮小します
拡大
ページを基準としてプロセス図を拡大します
縮小
ページを基準としてプロセス図を縮小します
右回りに回転
プロセス図を右回りに90度回転します
左回りに回転
プロセス図を左回りに90度回転します
微調整（左）*
プロセスをページの左側に移動します
微調整（右）*
プロセスをページの右側に移動します
微調整（上）*
プロセスをページの上側に移動します
微調整（下）*
プロセスをページの下側に移動します
ページセレクター
特定のページに移動できます
ズームドロップダウン
プレビューをズームインおよびズームアウトできます
単一ページビュー
一度に1ページずつ表示します
複数ページビュー
すべてのページを並べて表示します

*マウスでドラッグしてもプロセス図を移動できます。

印刷プレビューなしで印刷する

プレビューなしで印刷する方法は2つあります。これらのオプションは両方とも、デフォルトの用紙サイズを使用して、1ページの用紙に収まるようにプロセスページを拡大縮小します。


プロセスを印刷する
プロセス内のすべてのページを印刷するには、［ファイル］ -> ［印刷］ -> ［プロセス］の順に選択します。
ページを印刷する
現在のページを印刷するには、［ファイル］ -> ［印刷］ -> ［ページ］の順に選択します。
