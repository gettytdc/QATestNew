设置概述

配置数据网关需要以下步骤：

  1. 安装数据网关引擎组件

    数据网关引擎要求在运行数据网关流程的所有应用程序服务器所在的环境中安装多个组件。

  2. 启用数据网关流程

    必须启用数据网关流程，并在运行数据网关所需的每个应用程序服务器的“服务器配置详细信息”中选择用于与数据网关流程通信的端口。

  3. 创建数据网关 SQL 登录凭据和用户

    数据网关需要针对 Blue Prism 数据库所在的服务器实例创建 SQL 登录凭据和用户。

  4. 添加保存了 SQL Server 用户凭据的数据网关凭据

    必须使用 Blue Prism 凭据才能从数据网关引擎安全访问 Blue Prism 数据库。

  5. 配置 Blue Prism 数据网关设置

    这些设置用于确定数据网关引擎将处理哪些数据。

  6. 创建数据网关配置

    数据网关配置中的输出定义了将数据发送到的位置。这可以是平面文件、HTTP 端点、Splunk 实例或数据库。
