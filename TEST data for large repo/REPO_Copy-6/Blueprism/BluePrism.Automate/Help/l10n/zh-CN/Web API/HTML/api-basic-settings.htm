基本设置

API 定义中所有请求的通用组件和设置。

名称

用于在 Blue Prism 中标识 API 定义的名称（在环境中唯一）。在操作环节中选择 API 定义时使用该名称。

基本 URL

用于 API 中每个 HTTP 请求的 URL 的一部分。

高级设置

为已配置的 Web 服务应用连接超时设置：

  • HTTP 请求连接超时—Blue Prism 在引发异常之前等待对 HTTP 请求的响应的最长时间。
  • 授权服务器请求连接超时—Blue Prism 在引发异常之前等待对授权服务器请求的响应的最长时间。此设置仅可用于在 API 定义的通用身份验证设置中选中的 OAuth 2.0（客户端凭据）和 OAuth 2.0（JWT 持有者令牌）身份验证类型。
