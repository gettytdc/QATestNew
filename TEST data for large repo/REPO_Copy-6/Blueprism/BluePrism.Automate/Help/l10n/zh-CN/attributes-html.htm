应用程序属性—HTML 模式

简介

以下是 Windows 探测工具在 HTML 模式中返回的属性列表。有关由其他模式返回的属性列表，请参阅：—

  • Win32 模式
  • Active Accessibility 模式
  • UI Automation 模式
  • Java 模式

应用程序属性

X
元素左上角相对于其父元素的水平坐标。另请参阅 Y、宽度和高度。 每个坐标单位代表一个屏幕像素。
Y
元素左上角相对于其父元素的垂直坐标。另请参阅 X、宽度和高度。 每个坐标单位代表一个屏幕像素。
宽度
以屏幕像素为单位的元素宽度。 另请参阅 X、Y 和高度。
高度
以屏幕像素为单位的元素高度。 另请参阅 X、Y 和宽度。
路径
文档对象模型 (DOM) 中元素路径的文本表示。例如，路径 HTML/BODY(1)/TABLE(4) 是指在第一个主体元素（在 DOM 中的 HTML 元素下声明）下声明的第四个表元素。
TagName
表示元素的标记的名称，在 HTML 文档源中声明。例如，超链接的标记名称是“a”：超链接由定位标记构成，而定位标记由 HTML 中的字母 a 表示。
ClassName
表示元素的标记的类名，在级联样式表 (CSS) 中使用。例如，在 HTML 标记 <div class="navigation"> 中，类名是“navigation”。
ID
表示元素的 HTML 标记的 ID，在 HTML 源中指定。 例如，标记 <a id="Section3> 的 ID 是“Section3”。
标题
表示元素的 HTML 标记的标题。
链接
例如，超链接的目标 URL。
InputType
仅与输入窗体中的元素相关。此属性是要输入的数据类型。
InputIdentifier
仅与输入窗体中的元素相关。此属性是相关输入元素的唯一标识符。
值
元素中显示的值（如有）。仅与编辑字段、复选框或标签等元素相关。
已启用
指示是否已启用元素。未启用的元素无法接收键盘或鼠标输入。
已选中
指示是否选中元素。仅与表示 True/False、Yes/No 或 On/Off 值的元素相关，如输入窗体中的复选框。
