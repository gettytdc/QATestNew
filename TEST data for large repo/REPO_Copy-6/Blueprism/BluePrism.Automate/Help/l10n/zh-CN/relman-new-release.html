新建发行版向导

新建发行版向导

概述

新建发行版向导用于创建发行版—一个输出文件，包含在发行包中定义的所有组件，这些组件均处于其当前配置状态。

成功执行后，向导将生成一个文件，该文件可以导入另一个 Blue Prism 环境，从而将所有组件及其状态传输到目标环境中。

可从客户端的主菜单启动该向导—可通过选择“文件 | 导出 | 新建发行版”或按快捷键 Ctrl-R 来访问；也可以通过点击发行版管理器中的“新建发行版”任务或按下相关发行包详细信息面板中的“新建发行版”按钮来启动向导。

环节

创建发行版涉及的环节如下：—

发行包

选择要从中创建发行版的发行包—现有的发行包或仅用于此发行版的临时发行包。

注意： 如果您是从发行版管理器中的选定发行包创建发行版，则不会显示此环节。

发行包内容
确定应从中创建发行版的临时发行包的内容。只有在发行包环节中选择临时发行包时才显示。
名称
设置发行版的名称
发行版说明
设置发行版的发行版说明
输出文件
设置生成的发行版的输出文件目标

发行包

在此环节选择应从中创建发行版的发行包。

可以选择预先存在的发行包，或随即创建一个用于从中创建发行版的临时发行包。

请注意，发行版内容是从数据库加载的，而不是从客户端中当前打开的内容加载。如果尚未保存相关组件，则发行版可能不包含最新版本。

如果在发行版管理器中选择一个发行包并从此处启动向导，则省略此环节。

临时发行包内容

在此处配置从中创建发行版的临时发行包的内容。

如果由于此向导未设置为从临时发行版构建发行版（因为向导是从发行版管理器中的选定发行包启动的，或者因为在发行包环节选择了预定义的发行包），则省略此环节。

名称

在此处输入发行版的名称。名称必须与相同发行包中的其他发行版不同。

发行版名称为必填项—即必须至少包含一个字符，最多可包含 255 个字符。

发行版说明

每个发行版都可包含一组说明，其中包括特定于发行版的说明或文件记录段落。在此环节中输入这些说明。

这些说明不是必填项，并且对可输入文本数量没有实际限制。

选择发行版后，发行版说明将显示在发行版详细信息面板上。

输出文件

可以在此设置应将发行版保存到的目标文件。如果所选文件名已经存在，会发出警告。

输出文件环节将根据所选发行版名称预填充文件名，并保存到在发行版管理器向导中指定的最后一个目录中。

导出发行版

选择文件并按“下一步 >”之后，向导将加载发行包的所有组件并将它们导出到所选输出文件中。

如果导出时发生任何错误，将显示错误，并且用户可以选择返回并更改之前的任何环节（例如，选择不同的输出文件路径或不同的名称）。
