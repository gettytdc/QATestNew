Stage Types

Page Stage

The page reference properties dialog has a number of fields to fill in.

1.Enter a name

The name that you enter identifies the stage and will be displayed on the process diagram.

2.Enter a description

The additional description field allows a lengthier narrative to be entered.

3. Choose a page

From the drop down list select the page to be executed.

4. Edit Inputs

Edit the inputs of the page.

5. Edit Outputs

Edit the outputs of the page.

Input Parameters to the process are specified here. Correspondingly, if the start stage resides on a subpage, the parameters set relate to that page; any stage which references that subpage will have the opportunity to pass values via the page reference stage to the page itself. To add a parameter simply click the grid below the "name" column to add a name to the parameter. Then choose a datatype below the "data type" column. Finally select a data item to store the input parameter to by dragging a data item from the data explorer and dropping it into the value column.
