The Splunk configuration should be here:
%appdata%\splunk_config.xml

The format of the file configured for token authentication is as follows:
<config>
  <endpoint name="HTTP Server">
  <host>localhost</host>
  <port>8088</port>
  </endpoint>
  <flag name="Secure">false</flag>
  <text name="Path">/services/collector/event</text>
  <text name="Token">26D5E976-8969-4D9D-B336-BE5711EE8C89</text>
  <text name="Source">BluePrism</text>
</config>

The format of the file configured for basic authentication is as follows:
<config>
  <endpoint name="HTTP Server">
  <host>localhost</host>
  <port>8088</port>
  </endpoint>
  <flag name="Secure">false</flag>
  <text name="Path">/services/collector/event</text>
  <text name="Username">Username</text>
  <text name="Password">Password</text>
  <text name="Source">BluePrism</text>
</config>
