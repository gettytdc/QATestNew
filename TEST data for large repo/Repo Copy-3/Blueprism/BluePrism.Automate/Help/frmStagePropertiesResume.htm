Resume Stage

The resume stage allows the process to resume normal flow of execution after recovering from and exception. For an overview of exceptions see: Exceptions Help

1. Enter a name

The name that you enter identifies the resume stage, it will be displayed on the process diagram.

2. Enter a description

The additional description field allows a lengthier narrative to be entered.
