Control Room

The Control Room area of Blue Prism allows processes to be started and stopped and their current status assessed. There are three entry points to the control room, accessed via the side bar, application main menu or top menu.

Start a Process

Start a pre-designed business process on a specific PC using a step-by-step wizard, for inexperienced users.

Stop a Process

Stop a single business process running on a specific PC using a step-by-step wizard, for inexperienced users.

Go straight to Control Room

Manage the Blue Prism run-time environment by starting and stopping multiple processes across multiple resources. The Control Room features a drag and drop user interface which allows experienced users to quickly define an environment and start processes running without requiring repetitive configuration.

More in-depth features such as logging levels and fault finding can also be determined within the Control Room.
