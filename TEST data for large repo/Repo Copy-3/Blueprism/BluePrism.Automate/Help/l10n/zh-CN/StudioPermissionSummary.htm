Studio 权限摘要

概述

以下权限可用于控制特定用户可以对 Studio（以及应用程序的其他部分）中的对象和流程执行的操作。有关如何通过用户角色为用户分配权限的详细信息，请参阅用户权限。

权限权利
创建 用户可以在 Studio 中创建项，包括在编辑器中使用“另存为”选项。 假定还具有“编辑”、“执行”和“查看”权限。
编辑 用户可以在 Studio 中编辑项，并通过 AutomateC 发布/取消发布流程。 假定还具有执行和查看权限。
执行 用户只能执行项（即，他们不能在调试自己的对象或流程时步入项）。“执行”权限不会自动授予“查看”权限，需要打开对象/流程的用户应当具有“查看”或“编辑”权限
查看 用户具有对项的只读查看权限。例如，右键点击控制室或使用对象/流程比较工具。如果用户还具有“执行”权限，那么他们可以在调试自己的对象或流程时步入项。
删除 用户可以从 Studio 中删除项
导出 用户可以从“文件”菜单、对象/流程编辑器、AutomateC 或“发行版”选项卡导出项（假设他们可以访问它）
导入 用户可以从“文件”菜单、AutomateC 或“发行版”选项卡导入项（假设他们可以访问它）
编辑组 用户可以在 Studio 树中创建/重命名/移动/删除组
