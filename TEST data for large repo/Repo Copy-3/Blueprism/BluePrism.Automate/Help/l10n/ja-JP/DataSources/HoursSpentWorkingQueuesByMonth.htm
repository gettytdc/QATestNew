Blue Prism

月別のワークキューの経過時間

BPDS_HoursSpentWorkingQueuesByMonth

デジタルワーカーが1つまたは複数のワークキューに費やした合計時間を月ごとに返します。

ソース：Blue Prismデータベース - ライブテーブルとMIリポジトリ（有効にするにはMI統計のコンパイルが必要です）

サンプルチャート

パラメーター

パラメーター名 データ型 説明 検証
@NumberOfMonths 整数 現在の日付を基準に統計を表示する月数で、指定されていない場合、デフォルトは6です。 1以上
@QueueName テキスト 統計を表示する特定のワークキューの名前で、指定されていない場合、デフォルトはワークキューすべてです。 なし。
