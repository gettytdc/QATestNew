スキルステージ

Blue Prismインテリジェントオートメーションスキルは、外部のサービスや機能への事前設定された接続など、ブランド化されパッケージ化された機能アイテムで、Blue Prismプロセスオートメーション内でインポートして使用できます。

［スキルプロパティ］ダイアログには、入力するフィールドがいくつかあります。

1.名前を入力

入力した名前によってアクションが識別され、プロセス図に表示されます。

2.説明を入力

追加の説明フィールドには、さらに長い説明を入力できます。

3.アクションを選択

アクションは事前に選択されており、変更できません

ドロップダウンリストから実行する操作を選択します。

4.入力を編集

ドロップダウンリストからデータ型を選択し、アクションの入力を編集します。式を入力するか、データエクスプローラから値フィールドにデータアイテムをドラッグします。

5.出力を編集

データエクスプローラから値フィールドにデータアイテムをドラッグし、アクションの出力を編集します。
