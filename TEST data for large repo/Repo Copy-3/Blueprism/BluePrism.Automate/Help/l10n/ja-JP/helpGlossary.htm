アクティブ化（ウィンドウ）

ウィンドウのアクティブ化とは、ウィンドウをフォアグラウンドに移動してデスクトップのメインウィンドウにすることを意味します。アクティブ化されたウィンドウはキーボード入力を受け取るウィンドウで、通常はフルカラーのタイトルバーが表示されますが、他のすべてのウィンドウではグレーのタイトルバーが表示されます。

フォーカス（ウィンドウまたは要素）

フォーカスされる要素とは、特定のアプリケーションのためにキーボード入力を受け取る要素です。 テキストフィールドがフォーカスされている場合、そのフィールドには通常カレットとして知られる点滅カーソルが含まれます。ボタンなどの他の要素は、フォーカスがあるときは通常、点線の矩形で囲まれます。Tabキーを押すと、フォーカスをある要素から別の要素に移動できます。

フォーカスという言葉がウィンドウに関連して使われることがあります。そのような場合は、意味はよく似ていますがアクティブ化という語を代わりに使用する必要があります。
