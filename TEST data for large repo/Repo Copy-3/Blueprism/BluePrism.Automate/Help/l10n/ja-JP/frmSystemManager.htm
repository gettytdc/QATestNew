システム

概要

システムマネージャーモジュールでは、Blue Prismのインストールを管理の観点から保守できます。主に、システムマネージャーでは、ロックの維持やログファイルと統計データの管理などのデータベースアクティビティに加えて、ユーザーや役割の割り当てなどのBlue Prismシステムのさまざまな側面を構成できます。

システムマネージャーには、すべてのユーザー、プロセス、テストシナリオ、レポートをすばやく表示して管理する機能もあります。

システムマネージャーインターフェイスはアクティビティベースです。これにより、システム領域を表すアイコンを選択すると、そのコンポーネントに関連する情報とウィザードが表示されます。

たとえば、［ユーザー］を選択すると、現在構成されているすべてのユーザーが表示され、［ユーザーを作成］、［ユーザーを削除］、［ユーザーを編集］などの操作に加え、Blue Prismの役割のメンテナンスも可能になります。
