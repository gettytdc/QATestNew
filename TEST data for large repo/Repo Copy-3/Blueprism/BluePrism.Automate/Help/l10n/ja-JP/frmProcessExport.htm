プロセスエクスポート

注意：このメカニズムは、大部分がリリースマネージャーに置き換えられました。リリース内でプロセスをエクスポートする方法の具体的な詳細については、「新規リリースウィザード」のヘルプ文書を参照してください。

エクスポートプロセス機能を引き続き使用し、以前（4.1より前）のバージョンのソフトウェアで管理されているBlue Prism環境にプロセスとビジュアルビジネスオブジェクトをエクスポートできます。

概要

プロセスエクスポート機能を使用すると、現在のBlue Prismデータベース内にプロセス（またはビジュアルビジネスオブジェクト）のローカルコピーを作成できます。 これはXMLファイル—（人間による検査とメール転送に便利なフォーマット）として保存されます。

プロセス/ビジネスオブジェクトをエクスポートする理由は次のとおりです。

  • テストデータベースからライブデータベースにコピーするため（テストデータベースからエクスポートした後、対象データベースにインポートする）。
  • テクニカルサポートのためにBlue Prismに送信するため。

プロセスをエクスポートする

プロセス（またはビジュアルビジネスオブジェクト）をエクスポートするには、次のステップを実行します。

  • プロセススタジオが起動したら、メインのドロップダウンメニューから［ファイル］を選択します。
  • ［エクスポート］機能を選択して、プロセスエクスポートウィザードを開きます。
  • 必要に応じて［プロセス］または［ビジネスオブジェクト］を選択し、［次へ］をクリックします。
  • リストから対象アイテムを選択し、［次へ］をクリックします。
  • エクスポートしたプロセスを保存するファイルの場所を入力します。これは新しいファイル名にすることも、既存のファイルを上書きすることもできます。
  • ［終了］をクリックして操作を完了します。既存のファイルを上書きすることを選択した場合は、確認のメッセージが表示されます。ここでは、気が変わった場合に代わりに別のファイル名を選択できます。
