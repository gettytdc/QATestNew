ヘルプ

プロセスをテストする

テスト機能を使用すると、プロセスを変更できないことを条件に、テスト目的でプロセススタジオにアクセスできます。ユーザーはプロセス内を移動してプロセスをデバッグすることはできますが、変更を加えることはできません。

プロセスを見つける

データベースから、名前、バージョン、説明の詳細が一覧表示された既存のプロセスのリストが表示されます。適切な見出しをクリックすると、ボックス内の情報を各列に従ってソートできます。テストするプロセスを選択するには、マウスを使用して必要なエントリを強調表示して［次へ］ボタンをクリックします。

プロセススタジオに進む

ステップ1が完了したら、「OK」をクリックして選択したプロセスを開き、テストの開始準備ができたキャンバスにプロセスを読み込み、プロセススタジオに進みます。
