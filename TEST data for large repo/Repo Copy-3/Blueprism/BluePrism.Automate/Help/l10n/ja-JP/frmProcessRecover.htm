プロセスバックアップを回復

Blue Prismの自動バックアップ機能の概要については、自動保存のヘルプページを参照してください。

1.バージョンを選択する

システム障害の影響を受ける各プロセスには、元のバージョンと最新のバックアップバージョンがあります。回復するバージョンを選択します。バージョンかわからなくても心配しないでください。この操作は元に戻すことができます（再度回復フォームにアクセスして別のバージョンを選択します）。

2.復元する

［復元］ボタンをクリックします。この時点で、次回プロセススタジオを開くと、選択したバージョンのプロセスが表示されます。他のバージョンに戻す場合は、このフォームに戻ってそのバージョンを復元します。

目的のバージョンを選択したら、プロセススタジオにアクセスして［保存］ボタンをクリックします。これにより、選択したバージョンが保存され、既存のバックアップが消去されます。プロセスの編集は通常どおり続行できます。
