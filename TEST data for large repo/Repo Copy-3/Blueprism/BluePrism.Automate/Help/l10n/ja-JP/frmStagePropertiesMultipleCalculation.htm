この機能はバージョンの制限を受けます。詳細については、バージョンのページを参照してください。

複数の計算ステージ

複数の計算ステージでは、複数の計算をすべて1つのステージ内で順番に実行します。 有効な式を構築する方法の詳細については、「計算と決定」を参照してください。

プロパティのフォーム

［複数計算プロパティ］フォームは、式と結果の2つの列を持つリストで構成されています。［式］フィールドには、自由形式のテキストとして式を入力できます。また、高度な式編集のために［式編集］ボタンを押すと、標準の［式編集］ダイアログが表示されます。［結果］フィールドには、データアイテムの名前を入力できます。入力したデータアイテム名がまだ存在しない場合は、［データアイテムを作成］ボタンをクリックします。リストの右側には、データアイテムのツリービューが表示されます。アイテムをこのツリービューから［式］フィールドにドラッグして式の一部に使用したり、［結果］フィールドにドラッグしたりできます。

複数の計算ステージ内の行の順序は、実行される順序を示しています。実行順序を変更するには、フォームの下部にあるボタンを使用して行を上下に移動します。実行の前のステップの結果は、次のステップの式で使用できます。
