ヘルプ - シングルサインオングループリスト

概要

Blue Prismでは、シングルサインオン構成の一環として、ユーザーが属するActive Directoryグループを表示できます。特定のユーザーがBlue Prismへのアクセスに問題がある場合には、これが便利な診断ツールになることがあります - Blue Prismのユーザーの役割にマッピングされた、適切なグループメンバーシップを持っていることを確認できます。

使用法

この機能はシステムマネージャーの［ユーザー］セクションからアクセスでき、現在のデータベースがシングルサインオンに構成されている場合にのみ利用できます。

確認するユーザーを選択し、右側から［ユーザーのグループメンバーシップを表示］をクリックすると、2つのグリッドを含むフォームが開きます。最初のグリッドには、すべてのBlue Prismユーザー役割、各役割がマップされているActive Directoryグループ、ユーザーがそのグループのメンバーであるかどうかが表示されます。2番目のグリッドには、ユーザーがメンバーになっているすべてのActive Directoryセキュリティグループが表示され、その下にActive Directoryクエリが実行された場所の説明が表示されます。
