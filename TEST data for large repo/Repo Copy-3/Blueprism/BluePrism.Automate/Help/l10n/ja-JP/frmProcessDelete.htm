既存のプロセスを削除する

プロセスを削除する必要がある場合は、［削除］オプションでデータベースからプロセスを削除します。ただし、一度プロセスが実行されると監査目的で必要になるため、実行されていないプロセスのみを削除します。次のように画面上のステップに従ってプロセスを削除します。

1.プロセスを見つける

データベースから、名前、バージョン、説明の詳細が一覧表示された既存のプロセスのリストが表示されます。適切な見出しをクリックすると、ボックス内の情報を各列に従ってソートできます。

削除するプロセスを選択するには、マウスを使用して必要なエントリをハイライトします。

2.プロセスを削除する

ステップ1が完了したら、［OK］をクリックすると、選択したプロセスがデータベースから削除されます。ダイアログは開いたままになっているため、別のプロセスを削除できます。プロセスの削除が終了したら、［キャンセル］をクリックします。
