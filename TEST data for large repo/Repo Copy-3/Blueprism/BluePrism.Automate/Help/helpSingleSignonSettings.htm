Help - Single Sign-on Settings

Domain

You should enter the domain where the Blue Prism Security Groups will reside. You must enter the fully qualified domain name (e.g. "global.MyDomain.local").

Verify Connection

This button can be used to validate that Blue Prism can connect to a domain using the name provided from the current device.

Administrators Group

When configuring single sign-on for Blue Prism it is necessary to select an Active Directory security group whose members will be granted access to all areas of Blue Prism. The configuration of single sign-on for Blue Prism should be undertaken as a user who belongs to this security group.

Once the database is created and configured for single sign-on, a Blue Prism administrator should sign in to Blue Prism and configure the required Blue Prism security roles. Once these have been defined, each should be associated with an Active Directory security group which can be managed using the Manage Roles dialog.
