Help

The following is a list of elements which have no support for a given application type. What this means is that although the element may be spyable the read write and navigate actions will not work against these elements.

Visual Studio 2005 Based application

  • Spin buttons - Using Active Accessibility

Visual Basic 6.0 Based application

  • Label - Using Active Accessibility
  • Spin Button - Using Active Accessibility

Web based application

  • Label - Using Active Accessibility
  • List box - Using Active Accessibility
  • Combo box - Using Active Accessibility

Java Swing based application

  • Scrollbar - Has no write support
  • Trackbar - Has no write support

Java AWT based application

  • Listbox - Has no support
  • Scrollbar - Has no support
  • Menu - Has no support
  • Menu Item - Has no support
