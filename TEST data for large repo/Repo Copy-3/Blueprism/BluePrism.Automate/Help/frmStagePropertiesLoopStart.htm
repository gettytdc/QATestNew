Stage Types

Loop Start Stage

Loops are used in Blue Prism to sequentially access the contents of a collection. Typical scenarios include extracting customer data one by one and calculating total debts of multiple accounts by summing them one by one.

Data in a collection need not be accessed sequentially; if only a small number of items in the collection are desired at a time then each item in the collection may be referenced by name. See the help page on collections for an overview.

Completing the Properties Form

The loop start stage properties dialog has a number of fields to allow you to define what happens at the start of the loop. What happens during the loop will be determined by the business objects, the calculations etc that you place between the loop start and the loop end.

1. Enter a name

The name that you enter identifies the loop start, it will be displayed on the process diagram.

2. Enter a description

The additional description field allows a lengthier narrative to be entered.

3. Select Collection

The collection drop down allows you to select a collection that will be used as the basis for the loop.
