Help - Single Sign-on Groups List

Overview

Blue Prism allows you to view the Active Directory groups to which a user belongs, as part of a Single Sign-on configuration. This may prove to be a useful diagnostic tool in the event that a specific user has trouble accessing Blue Prism - you may check that they have the appropriate group memberships, as mapped against Blue Prism user roles.

Usage

This feature is accessed via the "Users" section in System Manager, and is only available when your current database is configured for Single Sign-on.

Select the user of interest, and click "View User's Group Membership" from the right hand side, which will open a form containing two grids. The first grid shows all Blue Prism user roles, the Active Directory group each role is mapped to and whether the user is a member of that group. The second shows all Active Directory security groups that the user is a member of with a description underneath of where the Active Directory query was perfomed.
