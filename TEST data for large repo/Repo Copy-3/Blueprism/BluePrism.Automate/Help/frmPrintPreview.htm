Print Preview

Print preview allows you to check print settings before printing out a process. As well adjusting printer and paper settings you can also scale, move and rotate a process to fit on the paper. The buttons and drop downs shown on the print preview are as follows.

Print
displays the print set up dialogue box
Page set up
displays the page set up dialogue box
Layout drop down
allows you to select the page layout to use for the preview
Scale to fit
increases the scale of the process diagram so that it fills the current page layout
Scale up
increases the scale of the process diagram relative to the page
Scale down
decreases the scale of the process diagram relative to the page
Rotate clockwise
rotates the process diagram 90 degrees clockwise
Rotate anti-clockwise
rotates the process diagram 90 degrees anticlockwise
Nudge left *
moves the process nearer the left of the page
Nudge right *
moves the process nearer the right of the page
Nudge up *
moves the process nearer the top of the page
Nudge down *
moves the process nearer the bottom of the page
Page selector
allows you to navigate to a particular page
Zoom drop down
allows you to zoom in and out of the preview
Single page view
displays one page at a time
Multi page view
displays all pages side by side

* Note that the process diagram can also be moved by dragging with the mouse.

Printing without Print Preview

There are also two ways to print without a preview. Both these options will scale process pages to fit on a single sheet of paper, using the default paper size.


Printing a Process
To print every page in the process select File -> Print -> Process
Printing a Page
To print the current page select File -> Print -> Page
