Create a New User

Four steps are required to complete the Create User wizard:

1. Enter a username

A unique user name is required for the new user. This must be no longer than 20 characters, but may contain spaces etc.

2. Enter a password

A start-up password must be entered for the new user. To maintain security, the user will be prompted to change their password when they first log in to the system.

3. Assign roles to the new user

See the User Roles help page for more information.

4. Create the new user

Selecting finish will create the user in the database with all of the selected attributes.
