Regular Expressions

Introduction

Blue Prism uses the regular expressions engine from the .NET framework. The regular expression patterns are defined by a special syntax or language, which is compatible with Perl 5 regular expressions.

The regular expressions engine is invoked with default options set.

Special Characters

The following characters are part of the regular expression syntax. To use any of the following characters in the regular expression as is they must be preceded by the escape character.

\Escape character.
[Character group start.
]Character group finish.
.Wild card, matches any single character except newline.
^The match must occur at the beginning of the string.
$The match must occur at the end of the string or before a newline.
(Group start.
)Group finish.
*Matches the previous element zero or more times.
+Matches the previous element one or more times.
?Matches the previous element zero or one time.
{Qualifier start
}Qualifier finish
|Matches any one element separated by the | character.
