应用程序属性—Java 模式

简介

以下是 Windows 探测工具在 Java 模式中（即仅适用于 Java 应用程序）返回的属性列表。 有关由其他模式返回的属性列表，请参阅：—

  • Win32 模式
  • Active Accessibility 模式
  • UI Automation 模式
  • HTML 模式

应用程序属性

活动
指示窗口当前是活动窗口。这包括窗口、对话框、框架等。 此外，此状态用于指示列表、表、树等组件的当前活动子项。 例如，列表的活动子项是使用矩形框起来的子项。
Armed
指示元素已调整。这通常用于已按下但尚未释放的按钮，而且鼠标指针仍悬停在按钮上方。
忙碌
指示当前元素处于忙碌状态。这通常用于进度栏、滑块或滚动条等元素，以指示它们处于转换状态。
已选中
指示当前已选中此元素。这通常用于切换按钮、单选按钮和复选框等元素，这些元素指示 On/Off、Yes/No 或 True/False 状态。
已折叠
指示此元素已折叠。此属性通常与可展开和已展开状态配对，并用于提供渐进式展开效果的元素（如树）。
可编辑
指示用户可以更改此元素的内容。 这通常主要用于允许用户输入文本的元素。如果启用滚动条和滑块等其他元素，则它们自动处于可编辑状态。
已启用
指示是否已启用此元素。未启用的元素不能由用户操作。在图形显示中，这样的元素通常是灰色的。
可展开
指示此元素允许渐进式展开其子项。此属性通常与层次结构元素（如树）一起使用，并且通常与已展开或已折叠状态配对。
已展开
指示此元素已展开。此属性通常与可展开和已折叠状态配对，并用于提供渐进式展开效果的元素（如树）。
可聚焦
指示此元素可以接受键盘焦点。可聚焦意味着在键盘上键入所产生的所有事件通常都会在此元素获得焦点时传递给它。 另请参阅：已聚焦。
已聚焦
指示此元素当前具有键盘焦点。已聚焦的元素是接收键盘输入的元素。 另请参阅：可聚焦。
水平
指示此元素的方向是水平而不是垂直。此属性与能以多个方向显示的滚动条和轨迹条等元素相关。
图标化
指示此元素最小化，并且仅由图标指示。此属性通常仅与框架和内框架相关联。
模态
指示必须先对此元素执行某些操作，然后用户才能与其他窗口中的元素进行交互。 此属性通常仅与对话框相关联。
多行
指示此（文本）元素可以包含多行文本。另请参阅单行。
可多选
指示此元素允许同时选择其多个子项。
不透明
指示此元素绘制其矩形区域内的每个像素。透明组件只绘制其部分像素，并允许其下方的像素“显露出来”。因此，不完全绘制其像素的组件提供一定的透明度。
已按下
指示当前已按下此元素。此属性通常与按钮相关联，指示用户在指针悬停于按钮上时按下了鼠标按钮且尚未释放鼠标按钮。
可调整大小
指示此元素的大小不固定。
可选择
指示此元素是允许选择其子项的元素的子项，并且此子项是可以选择的子项之一（如果用户需要）。
选定
指示此元素是允许选择其子项的元素的子项，并且此子项是已选择的子项之一。
显示
指示此元素、此元素的父项、此元素父项的父项等均可见。请注意，这并不一定意味着已在屏幕上绘制此元素；此元素可能被另一个显示的元素遮盖。
单行
指示此（文本）元素只能包含一行文本。这与多行元素形成对比。
暂时性
指示此元素是暂时性的。简而言之，这意味着此元素在用户界面中并未直接起作用，未能帮助用户界面中的一些其他元素。除等待查询外，暂时性元素通常不适用于自动化目的。
垂直
指示此元素的方向是垂直而不是水平。此属性通常与能以多个方向显示的滚动条和轨迹条等元素相关联。
可见
指示此元素可见。请注意：这意味着元素是特意可见的；但是，它实际上可能并未显示在屏幕上，因为它的一个包含元素不可见。
