Web 连接设置

配置在发出 Web 请求时应用的高级连接设置。任何为其用户角色启用“系统—Web 连接设置”权限的用户都可以配置这些设置。

可以为所有 Web API 请求配置默认设置。如果为特定的 URI 设置了值，则会覆盖默认设置：

  • URI—将为其应用设置的 URI。URI 必须仅由方案和主机组成，并且设置将应用于使用输入的 URI 的所有请求。
  • 连接限制—每个服务点的连接数。
  • 连接租约超时—允许连接在关闭之前保持空闲的时间。
  • 最大空闲时间—经过此秒数后关闭连接。
