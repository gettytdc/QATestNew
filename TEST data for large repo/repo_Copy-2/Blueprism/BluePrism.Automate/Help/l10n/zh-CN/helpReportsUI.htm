报告

计划报告

概述

报告提供了在特定日期范围内已执行的有效计划的筛选列表的视图。

在计划管理器树中选择一个报告后，它将显示在 Blue Prism 客户端的右侧窗格中。

从这里，可以设置报告的参数—对其进行报告的计划、要查看的日期范围以及报告本身。

安装 Blue Prism 时将会创建名为“最近活动”的默认报告。它将提供一个已在昨天或今天执行的所有计划的列表。

请注意，如果某个计划被取消或删除，其将不会出现在任何报告中。

创建报告

右键点击计划管理器树的“报告”元素并选择“新建报告”，即可创建报告。

将创建一个新报告，并用默认值填充。请注意，在报告被“刷新”之前（这会隐式将报告保存到数据库中）不会填充实际的报告数据。

配置报告

设置日期范围

您可以将报告设置为显示结束于指定日期的任意天数的已执行计划。

指定的结束日期可以设置为“今天”或“昨天”—每当运行报告时，这将使用当前日期或前一个日期。通过选中“今天/昨天”下拉列表旁边的单选按钮，可以启用此功能。

也可以通过选中日期选择器旁边的单选按钮并从中选择要使用的日期，将其设置为固定日期。

选择计划

报告可以列出任何或所有活动或过期的计划。

当列表中的相邻复选框被勾选时，计划会包含在报告中。

如果勾选了“所有计划”复选框，报告将动态包含所有计划。 请注意，这与单独勾选每个计划的不同之处在于，如果勾选了“所有计划”，报告将自动包含任何新计划，而其他情况则需要手动勾选复选框。

查看报告

首次在计划管理器树中选择报告以及保存报告时，将运行报告。

它可以通过点击“刷新”按钮手动运行—这会隐式地将报告保存到数据库中。

报告本身包含在指定日期范围内出现的计划激活列表。它将显示计划名称、计划执行的时间、计划完成的时间（如果没有记录结束时间—通常是计划仍在运行，则为空）、运行计划的服务器实例的主机名和端口，以及指向计划日志和计划本身的链接。

点击“查看日志”链接时，将打开该实例的计划日志。

点击“查看计划”时，会在计划管理器树中选中相关的计划。

导出报告

点击已打开的报告的“导出”按钮，可以将报告导出到文件中。

导出向导接受要将向其导出日志的文件以及导出数据的格式，可以是“文本”（人类可读格式）或“csv”（逗号分隔的变量格式，通常更适合在 Excel 等中查看）。

保存报告

可以通过在计划管理器中右键点击报告并选择“保存”来保存报告。

此外，点击“应用更改”按钮时，所有发生更改的计划、报告和时间表都将被保存。

最后，当点击“刷新”按钮刷新报告时，报告将被隐式保存。

删除报告

可以通过在计划管理器中右键点击报告并选择“删除”来删除报告。
