帮助—单点登录组列表

概述

在配置单点登录时，Blue Prism 允许您查看用户所属的 Active Directory 组。在特定用户无法访问 Blue Prism 时，这是一个有用的诊断工具—您可以检查他们是否具有相应的组成员资格，例如根据 Blue Prism 用户角色映射。

使用

此功能可通过系统管理器中的“用户”部分访问，且仅当您的当前数据库配置为单点登录时可用。

选择感兴趣的用户，然后从右侧点击“查看用户的组成员资格”，将打开一个包含两个网格的表格。第一个网格显示所有 Blue Prism 用户角色，每个角色映射到的 Active Directory 组以及用户是否属于该组的成员。第二个网格显示用户所属的所有 Active Directory 安全组，其中包含执行 Active Directory 查询的位置描述。
