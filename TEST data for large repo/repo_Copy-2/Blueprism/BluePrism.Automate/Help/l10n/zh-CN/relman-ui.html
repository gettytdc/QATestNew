发行版

概述

发行版管理器

发行版管理器界面直接位于欢迎屏幕之外，通过在客户端窗口左侧的“转到”列表中选择“发行版管理器”，可以随时在 Blue Prism 客户端中访问它。

通过此界面，您可以创建、删除和修改发行包，创建、导入和验证发行版，以及查看之前创建和导入的发行版的详细信息。

发行版管理器树

发行版管理器面板左侧的树由当前环境中注册的所有发行包及其任何相关发行版组成，而无论它们是在此环境中创建的还是从其他环境导入的。

从此环境创建的发行版在其图标中由从闭合包指向图标外部的绿色箭头表示。导入此环境的发行版包含一个从图标外部指向框的蓝色箭头。

在进入发行版管理器时，或者如果选择了树的根条目，则发行包概述将显示在客户端右侧的详细信息面板中。

在树中选择发行包或在发行包概述中双击发行包时，详细信息面板中将显示发行包详细信息面板。

在树中选择发行版或在发行包详细信息面板中双击发行版时，详细信息面板中将显示发行版详细信息面板。

发行包概述

发行包概述会显示一个发行包列表，其中包含有关发行包的一些基本详细信息—发行包的名称、创建日期、创建发行包的用户、从该发行包创建上一个发行版的日期以及它是创建的还是导入的。

发行包概述面板中有一个按钮，可以启动“新建发行包”向导，以创建和配置新的发行包。

双击此概述面板中的一个发行包将在发行包详细信息屏幕中打开该发行包，并且选择发行包会启用特定于发行包的任务，例如，允许修改或删除该发行包。

发行包详细信息

“发行包详细信息”面板会显示有关发行包的所有详细信息，包括其名称、描述、创建日期、创建用户、发行包内容列表以及发行包的基本发行版日志。

发行包详细信息屏幕提供用于启动向导的按钮，用于编辑发行包或基于发行包创建新发行版。

双击“发行版”列表中的一个发行版将在发行版详细信息面板中打开该发行版，以替换此发行包详细信息面板。

发行版详细信息

“发行版详细信息”面板会显示发行版的所有详细信息—其名称、发行版说明、创建或导入日期及其创建或导入用户以及发行版内容（可能与发行包有所不同，例如，在创建发行版后修改发行包时或者删除或重命名了一些发行包内容时）。

发行版管理器—启用所有任务

发行版管理器任务

可以通过点击 Blue Prism 客户端最左侧的任务链接来执行发行版管理器任务。

如果未选择任何发行包或发行版，则只能执行那些不在特定发行包上执行的任务。通过在树或发行包概述中选择发行包或发行版，将启用特定于发行包的任务。

这些任务可以让您启动对应于所有发行版管理器功能的向导，即：—

  • 新建发行包
  • 修改发行包
  • 删除发行包
  • 新建发行版
  • 导入发行版
  • 验证发行版
