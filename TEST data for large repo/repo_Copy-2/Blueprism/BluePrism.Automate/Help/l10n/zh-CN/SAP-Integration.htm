帮助—SAP 集成

概述

SAP 集成由 Blue Prism 使用 SAP GUI 脚本 API 提供。该 API 由 SAP AG 提供，旨在实现其胖客户端平台 Netweaver 的自动化。

使用

必须先在 SAP 服务器上启用 GUI 脚本 API，然后 Blue Prism 才可以与 SAP 交互。要启用 SAP GUI 脚本，请打开 SAP 客户端并使用管理凭据登录。启动 RZ11 事务。在“维护配置文件参数”窗口中键入“sapgui/user_scripting”，然后点击“显示”。点击“显示配置文件参数属性”窗口中的“更改值”，然后在“新值”字段中键入 TRUE。 保存设置并从 SAP GUI 注销。

受支持的控件

SAP 按钮

该按钮控件支持一个导航操作“按下”。可以使用读取操作获取边界和文本。此控件不能写入。

SAP 复选框

标准复选框控件支持读取和写入值。

SAP 组合框

组合框项可以使用“选择项”操作通过索引或名称选择。 项数可以使用读取环节读取。可以读取所选文本，并且可以使用“获取所有项”来读取组合框中的完整项列表。

SAP 网格视图

网格视图支持将网格的全部内容读入集合。仅显示基于文本的值—包含图标或类似内容的列将显示为空。

SAP 标签

标签控件支持通过读取环节获取文本。

SAP 菜单栏

SAP GUI Scripting API 中的怪异模式是指这些内容只能在主 SAP 窗口最大化时才可进行探测，尽管它们在探测后会正常运行而不管最大化状态如何。与其他应用程序类型一样，它们支持按路径选择菜单项—示例路径为“{File}{Save}”。

SAP 滚动条

sap 滚动条支持使用读取环节获取最大值、获取值和获取最小值，设置值导航操作以及设置值的写入。

SAP 滚动容器

sap 滚动条容器支持与 SAP 滚动条类似的一组操作。 不同之处在于每个操作都具有水平和垂直风格，并且没有写入环节功能。

SAP OkCodeField

确定代码字段是一个特殊的组合/文本输入框，它位于工具栏中并可以输入“SU11”等事务代码。输入代码会将用户带到应用程序中的目标位置。支持使用写入环节在确定代码字段上设置文本。 必须模拟按 Enter 键才能激活它。

SAP 密码字段

可以使用写入环节写入 sap 密码字段。

SAP 单选按钮

可以写入和选择 sap 单选按钮。还可以读取文本和当前状态。 请注意，写入时，该值不能设置为 False。因为这会破坏单选设置。 尝试将值设置为 false 会引发异常。

SAP 表控件

表控件支持将表的全部内容读入集合。 仅显示基于文本的值—包含图标或类似内容的列将显示为空。

SAP 选项卡

sap 选项卡控件表示选项卡条中的单个选项卡。这些控件可以直接探测，而且提供“选择”操作来选择选项卡。

SAP 选项卡条

选项卡条控件是一组选项卡的父级。它不能直接探测，但可以从模型中选择。可以按名称选择项。

SAP 文本字段

标准编辑字段。支持读取和写入。

SAP CTextField

ctextfield 控件与文本字段相同，只是它添加了一个访问最近使用的条目的组合按钮。

SAP 树视图

Sap 树视图支持读取所选节点的“键”，并获取给定键的节点文本。按顺序将二者组合意味着可以获得所选的节点文本。这可以作为单个读取环节的两个步骤来完成。标准的“选择项”操作提供直接的一步操作。 与其他应用程序类型中的树视图不同，SAP 树节点不能附加复选框。因此，SAP 树视图未实现“将项设置为已选中”。无法确定 SAP 树节点的展开或折叠状态，因此无法实现“切换项的展开状态”，无法提供任何读取该状态的方法，也无法对展开/折叠功能进行自动测试。无法通过 SAP API 实现“确保项可见”。选择树节点时，其父节点已经展开—API 不会自动为您展开，如果节点被隐藏，将无法找到它。

HTML 查看器

此外，SAP API 提供有基础 HTML 文档的视图，Blue Prism 可以集成通过 HTML 探测模式访问的这些控件。

不受支持的控件

  • GuiBarChart
  • GuiCalendar
  • GuiChart
  • GuiColorSelector
  • GuiContextMenu
  • GuiCustomControl
  • GuiDialogShell
  • GuiEAIViewer2D
  • GuiEAIViewer3D
  • GuiMap
  • GuiMenu
  • GuiSapChart
  • GuiNetChart
  • GuiPicture
  • GuiTitlebar

扩展支持

可以使用基于文本的配置文件配置 SAP 连接器。有关如何修改这些配置文件的说明，不在本帮助文档的范围之内。请联系 Blue Prism 支持，了解有关配置文件的使用信息。
