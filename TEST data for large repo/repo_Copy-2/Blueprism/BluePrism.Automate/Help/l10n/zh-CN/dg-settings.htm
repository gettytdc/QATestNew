配置数据网关设置

通过确定将处理哪些数据、在何处处理相关数据以及发送数据的频率来配置数据网关引擎。

  1. 选择“系统”选项卡。

  2. 选择数据网关>设置。

  3. 选择所需选项，确定会话日志数据的存储位置（至少需要选中一个选项）：
    • 将会话日志写入数据库—会话日志将会发送至 Blue Prism 数据库的会话日志表。如果未选择，则在 Blue Prism 客户端中查看新会话日志的功能将不可用。
    • 将会话日志发送到数据网关—会话日志将发送到 Blue Prism 数据库上的临时存储表，数据网关引擎将在该表中访问这些日志，以便在配置的输出中使用。
    • 如果处理数据的过程中无法访问指定的 HTTP 或数据库数据网关端点，则在该端点恢复可用之前，系统会暂时将这些数据存储在“死信队列”中。
  4. 选择将已发布仪表盘发送至数据网关，来将数据从已发布的仪表盘发送到数据网关系统中的数据库表，并设置为每个仪表盘发送数据的频率。

  5. 选择将工作队列分析快照数据发送到数据网关，以将工作队列分析数据发送至数据库。

  6. 将“状态监控频率”设置成 5 到 3600 秒之间的值。这决定了“控制室”屏幕中“数据网关”屏幕的刷新频率。

    有关 Blue Prism 仪表盘的更多信息，请参阅产品内帮助中的仪表盘概述主题。

  7. 点击“应用”保存设置。
