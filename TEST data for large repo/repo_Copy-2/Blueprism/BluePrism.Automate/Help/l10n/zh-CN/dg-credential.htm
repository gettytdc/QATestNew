数据网关凭据。

数据网关凭据可用于任何需要经过身份验证访问数据库或 HTTP 端点的数据网关配置。除此之外，还必须创建数据网关凭据来存储 SQL server 用户帐户，该帐户为数据网关引擎提供对数据库的访问权限。

“数据网关”凭据类型的凭据只能由应用程序服务器在生成数据网关配置时访问，它们不能用于标准 Blue Prism 流程或其他 Blue Prism 元素（如 Web API 定义）。

为 SQL Server 用户添加凭据

创建凭据来存储必要的 SQL Server 用户帐户，以便安全访问 Blue Prism 数据库。

凭据名称、用户名和密码必须与下面指定的各项完全匹配，以便反映为 SQL Server 登录输入的详细信息。

  1. 选择系统选项卡。

  2. 选择安全>凭据。

  3. 点击新建。此时将显示“凭据详细信息”窗口。
  4. 配置以下凭据：
    • 名称—数据网关 BP 数据库 SQL 用户
    • 类型—数据网关凭据
    • 用户名—数据网关 BP 数据库 SQL 用户
    • 密码—为 SQL Server 用户指定的密码。

  5. 点击确定保存凭据。
