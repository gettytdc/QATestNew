スキル管理

この画面には、インポートされたすべてのスキルの最新バージョンが表示され、これらを有効または無効にするコントロールが提示されます。有効になっていないスキルは、オブジェクトやプロセスのステージとしてすでに使用されているプロセスを含め、システム内の任意の場所で使用できません。オブジェクトスタジオとプロセススタジオのスキルツールバーにも表示されません。有効になっていないWeb APIコンポーネントにリンクされているスキルは自動的に無効になり、［Web API］列に警告アイコンが表示されます。

以前にインストールしたスキルのバージョンを表示するには、該当するスキルの［以前のバージョン］列にあるリンクをクリックします。

スキルが使用されているオブジェクトとプロセスのリストを表示するには、該当するスキルの［参照を検索］リンクをクリックします。
