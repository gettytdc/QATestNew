アナリティクス

ダッシュボードの概要

ダッシュボードは一連のタイルで構成され、各タイルはさまざまなデータセットから派生した情報を視覚的に表現します。次の種類のダッシュボードが利用可能です。

  • パーソナル：適切な許可を持つユーザーがすべてを構成できますが、作成したユーザーにのみ表示されます。
  • グローバル：すべてのユーザーに表示されますが、特別に割り当てられた権限を持つユーザーのみが構成できます。
  • パブリッシュ：Blue Prismクライアント内に表示されるのではなく、外部アプリケーションとデータを共有するために使用されます（特別に割り当てられた権限を持つユーザーのみが構成できます）。

デフォルトダッシュボードは自動的にインストールされ、特に代替する方法を設定していないユーザーのためにホーム画面に表示されるダッシュボードです。デフォルトダッシュボードは、変更はできますが削除ができない特別なグローバルダッシュボードです。

ダッシュボードを使用する

ダッシュボードは基本的に保存済みのタイルコレクションのビューです。適切な許可が付与されている場合、ユーザーは［ダッシュボード］タブ内で既存のダッシュボードを作成および編集できます。

ダッシュボードを作成する

新しいダッシュボードは、［ダッシュボード］セクションの［ダッシュボード］タブから作成できます。右クリックオプションの［パーソナルダッシュボードを作成］、［グローバルダッシュボードを作成］、または［パブリッシュされたダッシュボードを作成］を選択すると、パーソナルダッシュボード、グローバルダッシュボード、またはパブリッシュされたダッシュボードのいずれかが新規作成されます。

  • パーソナルダッシュボード
    • ログインユーザーによって作成され、そのユーザーにのみ表示されるダッシュボードです
    • ダッシュボード許可の「パーソナルダッシュボードを設計」は、このダッシュボードタイプへのアクセスを制御します
  • グローバルダッシュボード
    • アナリティクスセキュリティグループの適切な許可を持つすべてのユーザーが利用できるダッシュボードです
    • ダッシュボード許可の「グローバルダッシュボードを設計」は、このダッシュボードタイプへのアクセスを制御します
  • 公開されたダッシュボード
    • ユーザーのホームページとして設定できないダッシュボードです
    • タイルデータは選択したBPサーバーインスタンスによって定期的に取得され、イベントログを介して外部のアプリケーションで利用できます
    • ダッシュボード許可の「パブリッシュされたダッシュボードを設計」は、このダッシュボードタイプへのアクセスを制御します
    • 有効にすると、BPサーバーは30秒ごとにパブリッシュされたダッシュボードを確認します。更新間隔が最も長いタイルを使用してダッシュボード上のすべてのタイルを更新するかどうかを決定し、その結果をWindowsイベントログに出力します。

ダッシュボードを編集する

ダッシュボードを編集するには、適切なダッシュボード許可をユーザーの役割に割り当てる必要があります（「許可」を参照してください）。

ダッシュボードパネルでダッシュボードを選択すると、正しい許可を持つ役割を右クリックして表示されるコンテキストメニューから［編集］オプションが利用可能になります。

この［編集］機能を選択すると、選択したダッシュボードが［編集モード］になり、ダッシュボードコンテンツの設計を変更できるようになります。

ダッシュボード編集モード

編集モードになると、ダッシュボードに縦線が表示され、ダッシュボード名フィールドの境界線がハイライトされます。ダッシュボードが編集モードの間は、ダッシュボード以外では操作できません。ダッシュボードが保存されるか編集がキャンセルされるまで、ダッシュボードは編集モードのままになります。

ダッシュボードを編集している間は、次の操作を実行できます。

  • タイルを追加する
  • タイルをサイズ変更および削除する
  • ダッシュボード内でタイルを移動する
  • ダッシュボードタイトルを追加する
  • ダッシュボードを保存する
  • 編集モードをキャンセルする

次の操作は、ダッシュボードが編集モードになっている間のみ実行できます。コピーはダッシュボードが編集モードの間は実行できません。

ダッシュボードにタイルを追加する

ダッシュボードにタイルを追加するには、［タイルライブラリ］タブを選択して利用可能なタイルのリストを表示します。ダッシュボードにタイルをドラッグアンドドロップします。タイルは、現在何も表示されていない空白のセクションにドロップできます。

他の既存のタイルの片側にタイルをドロップした場合は、そのタイルが既存のタイルの位置に置き換わり、隣接するタイルは位置矢印で示された場所に移動します。 位置はタイルのサイズによって異なります。

タイルをサイズ変更および削除する

編集モードでタイルを右クリックすると、オプションを提示するコンテキストメニューが表示され、タイルのサイズを選択したり、タイルを削除したりできます。

  • 小 - このタイルはグリッドの1つの正方形を占めます
  • 中 - このタイルはグリッドの2つの正方形を占めます
  • 大 - このタイルはグリッドの1行を占めます
  • 削除 - 現在選択されているタイルがダッシュボードから削除されます
  • すべてのタイルを削除 - ダッシュボードのすべてのタイルが削除されます

ダッシュボード内でタイルを移動する

タイルはダッシュボードグリッド上の任意の位置にドラッグアンドドロップできます。小サイズのタイルは単一のグリッドスペースに収まりますが、中サイズと大サイズのタイルは2～3個のタイルスペースを必要とする場合があります。

タイルを移動すると、周囲のタイルはそのタイルに合わせて自動的に移動します。 タイルをダッシュボードにドロップする際、移動先のスペースをポイントする位置矢印の指示がドラッグ操作中に表示されます。

ダッシュボードにタイトルを追加する

ダッシュボードが編集モードになっている間、タイトル領域は編集可能になります。編集モードは、フィールドが白、境界線が青のハイライトで示されます。

ダッシュボードを保存する

これ以上編集する必要がなくなったら、ダッシュボードのドロップダウンメニューの［保存］ボタンを使用してダッシュボードを保存できます。編集中に加えられた変更が保持されます。［保存］ボタンをクリックしてもダッシュボードは［編集モード］になりません。 編集モードは、タイトルが現在選択されている配色のフルカラーに戻ることで示されます。

編集モードをキャンセルする

ダッシュボードのドロップダウンメニューで［閉じる］オプションを選択すると、ダッシュボードが閉じます。保存されていない変更がある場合は、現在の変更内容を破棄するかどうかを尋ねるプロンプトメッセージが表示されます。変更内容を破棄し、ダッシュボードの編集をキャンセルし、ダッシュボードを前回の保存状態に戻すには、［はい］を選択します。編集モードでダッシュボードの操作を続行するには、［いいえ］を選択します。

ダッシュボードをコピーする

ダッシュボードをコピーするには、現在編集していないダッシュボードを選択します。選択したら、右クリックして表示されるコンテキストメニューから［パーソナルダッシュボードとしてコピー］、［グローバルダッシュボードとしてコピー］、［パブリッシュされたダッシュボードとしてコピー］のいずれかの機能を選択します（「パーソナルダッシュボード」と「グローバルダッシュボード」を参照してください）。

ダッシュボードへの選択が済むと、コピーされたダッシュボードは適切なダッシュボードグループに配置され、ダッシュボード名のコピーという名前が付けられます。コピーしたダッシュボードは編集モードで開きます。

ダッシュボードを削除する

ダッシュボードを削除するには、これを選択し、右クリックして表示されるコンテキストメニューから［ダッシュボードを削除］を選択します。

現在のホームページダッシュボードとしてすべてのユーザーに設定されているダッシュボードは削除できません。 デフォルトダッシュボードも削除できません。ダッシュボードが編集モードの間は削除できません。

ホームページを設定する

パーソナルダッシュボードかグローバルダッシュボードを選択して、ユーザーのホームページに表示できます。 パーソナルダッシュボードをホームページに選択すると、そのページは現在のユーザーにのみ表示されます。グローバルダッシュボードをホームページに選択すると、すべてのユーザーに表示されます。

新しいホームページのダッシュボードを設定するには、［ダッシュボード］パネルのリストからダッシュボードを選択します。右クリックして表示されるコンテキストメニューから［マイホームページとして設定］を選択します。そのダッシュボードはダッシュボードのリストの一番上に表示され、適切なユーザーのホーム画面に表示されます。

許可

ダッシュボード許可は、ユーザーがタイルを操作したり、グローバルダッシュボードやパーソナルダッシュボードを設計したりする機能を制御します。

許可設定は［システム］画面から利用でき、［ユーザー］の下の［セキュリティ］セクションに含まれます。［ユーザー］画面で、［ユーザーの役割を管理］リンクを選択して［役割を管理］構成画面にアクセスします。

次の許可があります。

  • タイルを作成/編集/削除 - ユーザーはタイルを作成/編集/削除/整理できます。
  • グローバルダッシュボードを設計 - ユーザーはタイルライブラリへの読み取り専用アクセス権を持ち、グローバルダッシュボードを作成/編集/削除できます。
  • パーソナルダッシュボードを設計 - ユーザーはタイルライブラリへの読み取り専用アクセス権を持ち、自分のパーソナルダッシュボードを作成/編集/削除できます。
  • パブリッシュされたダッシュボードを設計 - ユーザーはタイルライブラリへの読み取り専用アクセス権を持ち、パブリッシュされたダッシュボードを作成/編集/削除できます。
  • ダッシュボードを表示 - ユーザーは［ダッシュボード］タブにアクセスして利用可能なダッシュボードをクリックできます。自分のホームページに表示するダッシュボードを選択できます。タイルライブラリへのアクセス権はなく、ダッシュボードを作成/編集/削除することはできません。

どのダッシュボード許可もないユーザーには、Blue Prismにログインしたときにデフォルトダッシュボードが表示されます。
