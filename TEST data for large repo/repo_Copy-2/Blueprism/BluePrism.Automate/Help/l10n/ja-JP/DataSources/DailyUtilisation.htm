Blue Prism

日別使用率

BPDS_DailyUtilization

指定された期間におけるリソース全体の使用率、つまりリソースがアクティブなセッションの実行に費やした時間を返します。データは完了した日に対してのみ返されます（つまり、MI統計の最終コンパイル後に作成されたセッションは含まれません）。

ソース：Blue Prismデータベース - ライブテーブルとMIリポジトリ（有効にするにはMI統計のコンパイルが必要です）

サンプルチャート

パラメーター

パラメーター名 データ型 説明 検証
@BPResourceName テキスト 照会する個々のリソースの名前（指定しない場合、すべてのリソースが照会されます） なし。
@NumberOfDays 整数 レポーティング期間（日数）（指定しない場合、日数は7日になります） 最大31日
@DisplayUnits テキスト

可能な値は次のとおりです。

  • 秒 - 値は秒単位で返されます
  • 分 - 値は分単位で返されます
  • 時間 - 値は時間単位で返されます

  • 割合 - 値は利用可能な合計処理時間の割合として返されます（例：指定された日に5リソースで処理が実行された場合、合計処理時間は5 x 24（リソースx時間）= 120処理時間になります。したがって、50%という結果は指定された時間枠内で60時間の処理時間に達したことを示します）

指定しない場合、使用率の値は分単位で返されます

 秒、分、時間、または割合
@MaxResourceHours 整数 リソースの最大処理時間数。この値は使用率の計算（割合で表示）に使用されます。@DisplayUnitsを参照してください（指定しない場合は24時間が想定されます）。 最大24時間
