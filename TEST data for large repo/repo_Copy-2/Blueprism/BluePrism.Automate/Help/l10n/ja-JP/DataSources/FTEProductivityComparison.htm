FTE生産性の比較

BPDS_FTEProductivityComparison

指定された期間中に完了したケースの数を返します。人員を表す指標と比較します（任意）。比較するにはパラメーター値が必要です。データは完了した日に対してのみ返されます（つまり、MI統計の最終コンパイル後に完了したケースは含まれません）。

ソース：Blue Prismデータベース - ライブテーブルとMIリポジトリ（有効にするにはMI統計のコンパイルが必要です）

サンプルチャート

パラメーター

パラメーター名 データ型 説明 検証
@BPQueueName テキスト 照会する個々のキューの名前（指定しない場合、すべてのキューが照会されます） なし。
@NumberOfMonths 整数 レポーティング期間（月数）（指定しない場合、月数は6か月になります） 最大12か月
@FTEProductivity 小数点

比較用の1 FTEの日別生産性（つまり、1 FTEが1日に作業するケースの平均数）を表します。指定しない場合、完了したケースの数が返されます

 なし。
@FTECost 小数点

@FTEProductivityが指定されている場合にのみ適用可能で、比較用に1 FTEの総負担費用（日別）を表します。

 なし。
@DisplayAs テキスト

@FTEProductivityが指定されている場合にのみ適用可能で、可能な値は次のとおりです。

  • 割合 - 値はFTE生産性の割合として返されます
  • 数値 - 値は等価FTEの数として返されます
  • コスト - 値は同数のFTEに基づくコスト削減として返されます（@FTECostの値が必要です）

指定しない場合、値はFTE生産性の割合として返されます

 割合、数、またはコスト
