ステージタイプ

プロセスステージ

［プロセスプロパティ］ダイアログには、入力するフィールドがいくつかあります。

1.名前を入力

入力した名前によってプロセスが識別され、プロセス図に表示されます。

2.説明を入力

追加の説明フィールドには、さらに長い説明を入力できます。

3.プロセスを選択

ドロップダウンリストから、実行するBlue Prismプロセスを選択します。

4.入力を編集

ドロップダウンリストからデータ型を選択し、プロセスの入力を編集します。式を入力するか、データエクスプローラから値フィールドにデータアイテムをドラッグします。

5.出力を編集

データエクスプローラから値フィールドにデータアイテムをドラッグし、プロセスの出力を編集します。
