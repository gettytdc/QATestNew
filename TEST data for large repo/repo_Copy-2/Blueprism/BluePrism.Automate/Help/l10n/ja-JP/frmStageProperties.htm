ステージタイプ

アクションステージ

［アクションプロパティ］ダイアログには、入力するフィールドがいくつかあります。

1.名前を入力

入力した名前によってアクションが識別され、プロセス図に表示されます。

2.説明を入力

追加の説明フィールドには、さらに長い説明を入力できます。

3.ビジネスオブジェクトを選択

ドロップダウンリストから、プロセスがアクションを実行するために使用するビジネスオブジェクトを選択します。

4.アクションを選択

ドロップダウンリストから実行する操作を選択します。

5.入力を編集

ドロップダウンリストからデータ型を選択し、アクションの入力を編集します。式を入力するか、データエクスプローラから値フィールドにデータアイテムをドラッグします。

6.出力を編集

データエクスプローラから値フィールドにデータアイテムをドラッグし、アクションの出力を編集します。
