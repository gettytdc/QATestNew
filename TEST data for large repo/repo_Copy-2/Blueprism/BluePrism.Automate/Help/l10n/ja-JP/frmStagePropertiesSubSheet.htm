ステージタイプ

ページステージ

［ページ参照プロパティ］ダイアログには、入力するフィールドがいくつかあります。

1.名前を入力

入力した名前によってステージが識別され、プロセス図に表示されます。

2.説明を入力

追加の説明フィールドには、さらに長い説明を入力できます。

3.ページを選択

ドロップダウンリストから実行するページを選択します。

4.入力を編集

ページの入力を編集します。

5.出力を編集

ページの出力を編集します。

プロセスへの入力パラメーターはここで指定されます。同様に、開始ステージがサブページにある場合、設定されたパラメーターはそのページに関連しています。そのサブページを参照するステージは、ページ参照ステージを介してページ自体に値を渡すことができます。パラメーターを追加するには、［名前］列の下のグリッドをクリックしてパラメーターに名前を追加します。次に、［データ型］列の下のデータ型を選択します。最後に、データエクスプローラからデータアイテムをドラッグして値列にドロップし、入力パラメーターを保存するデータアイテムを選択します。
