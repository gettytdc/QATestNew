既存のプロセスを編集する

既存のプロセスをすばやく簡単に編集するには、プロセススタジオのメインメニューの［プロセスを編集］ウィザードを使用します。画面上の番号付きオプションで示されているように、以下のステップに従ってください。

1.プロセスを見つける

データベースから、名前、バージョン、説明の詳細が一覧表示された既存のプロセスのリストが表示されます。適切な見出しをクリックすると、ボックス内の情報を各列に従ってソートできます。

編集するプロセスを選択するには、マウスを使用して必要なエントリをハイライトします。

2.プロセススタジオに進む

ステップ1が完了したら、［OK］をクリックして選択したプロセスを開き、プロセスをキャンバスに読み込んで作業の開始準備ができた状態で、プロセススタジオに進みます。
