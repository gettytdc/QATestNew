プロセス

プロセスビューには、コントロールルームにパブリッシュされている利用可能なすべての業務プロセスのリストがプロセスグループに表示されます。

プロセスをリソースに適用するには、必要なプロセスをハイライトして、実行するリソースの上にドラッグします。これにより、環境領域に保留中のエントリが作成され、現在のすべてのプロセスとリソースの組み合わせが実行中、停止、あるいは保留中かどうかが表示されます。新しく追加されたプロセス開始の準備ができているかどうかを確認します。

時間が経過してもプロセスがまだ同じステージにある場合、そのプロセスは「警告」状態であることが表示されます。この期間はシステム構成で構成可能で、無効にすることもできます。
