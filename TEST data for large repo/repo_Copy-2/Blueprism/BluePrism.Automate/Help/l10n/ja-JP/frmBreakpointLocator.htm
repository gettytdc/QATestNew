区切り点を特定および管理する

区切り点ロケーターフォームを使用すると、区切り点を見つけて管理できます。区切り点は、チェックボックスの設定に応じて、左側のツリービューに名前順またはページ順に表示されます。

既存の区切り点に設定されている条件は、右側のボックスに表示されます。これは編集できません。

個々の区切り点を削除するには、ツリービューでアイテムを選択して［クリア］ボタンをクリックします。すべての区切り点を一度にクリアするには、［すべてクリア］ボタンをクリックします。これらの変更は元に戻せません。

区切り点が作成されたステージを確認する場合は、ツリービュー内のアイテムをダブルクリックするか、それを選択して［区切り点に移動］ボタンをクリックします。
