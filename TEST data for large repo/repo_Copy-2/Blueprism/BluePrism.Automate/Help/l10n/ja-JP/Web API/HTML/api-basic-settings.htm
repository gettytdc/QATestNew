基本設定

API定義内のすべてのリクエストに共通のコンポーネントと設定。

名前

Blue PrismでAPI定義を識別するために使用される、環境内で固有の名前。この名前はアクションステージでAPI定義を選択する際に使用されます。

ベースURL

API内のすべてのHTTPリクエストに使用されるURLの一部。

詳細構成

構成したWebサービスに接続タイムアウト設定を適用します。

  • HTTPリクエストの接続タイムアウト - 例外がスローされるまでにBlue PrismがHTTPリクエストへの応答を待つ最大時間。
  • 認証サーバーリクエストの接続タイムアウト - 例外がスローされるまでにBlue Prismが認証サーバーリクエストへの応答を待つ最大時間。この設定は、API定義の［共通認証］設定で選択されている場合に、OAuth 2.0（クライアント認証情報）およびOAuth 2.0（JWTベアラートークン）認証タイプに対してのみ利用可能です。
