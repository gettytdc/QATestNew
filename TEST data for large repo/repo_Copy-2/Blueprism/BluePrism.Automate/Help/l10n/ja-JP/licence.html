はじめに

Blue Prismを使用するには、有効なライセンスをインストールする必要があります。ライセンスはシステムマネージャーを介してデータベースにインストールされ、そのデータベースに関連付けられているBlue Prism環境全体に適用されます。

ライセンスをインストールする

Blue Prismからライセンスキーを受け取った後、次の手順でインストールできます。

  1. Blue Prismのインスタンスにログインします。
  2. システムマネージャーに移動します。
  3. ナビゲーションメニューの［ライセンス］ページを選択します。
  4. ［新規ライセンスをインストール］をクリックしてライセンスファイルを選択します。

ライセンスに関するサポート

ライセンスに関するアドバイスやサポートが必要な場合は、Blue Prismにお問い合わせください。
