简介

本文假定您熟悉与 Web 服务有关的概念，也熟悉“WSDL”和“URL”等相关术语。有关 Blue Prism 中的 Web 服务的概述，请参阅 Web 服务帮助页面。

添加 Web 服务

“添加 Web 服务”向导由三个或四个环节组成，具体则取决于所添加服务的性质。

1. 提供 WSDL 的 URL

请输入描述您所要添加的 Web 服务的 WSDL 文档的 URL。

2. 选择 Web 服务

如果上一步所提供的 WSDL 文档描述了多个 Web 服务，则系统将要求您从所列服务中选择一个；如果您想要添加所有这些服务，则必须重复使用此向导以逐一添加各个服务。

如果 WSDL 文档只描述了一个服务，则系统将自动跳过本步骤。

3. 选择您想要提供的方法

系统将显示一个列表，其中列出 Web 服务提供的方法。在该服务下方，勾选您想在 Process Studio 中提供的每个方法旁边的框。

4. 为服务选择一个名称

请为您的 Web 服务命名。这个名称可以任选；而且，它将显示在 Process Studio，以便您识别相应的 Web 服务。
