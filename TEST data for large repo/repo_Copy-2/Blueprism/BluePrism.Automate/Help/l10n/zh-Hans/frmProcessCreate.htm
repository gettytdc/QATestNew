创建新流程

使用“创建流程”向导轻松、快捷地创建新的自动流程。在 Process Studio 中开始工作前，需要填写两项信息—在窗体中填写每项信息，如下所示：

1. 输入流程名称

必须为新流程指定名称。名称可以包含任何字符（包括空格），但长度不能超过 128 个字符。

2. 输入流程描述

业务流程的描述可以是任意长度，并将显示在已完成的流程图顶部的备注字段中。描述最长只能为 1000 个字符。

3. 继续操作 Process Studio

完成步骤 1 和 2 后，点击“完成”将创建新流程并进入 Process Studio，而且新流程将加载到准备开始工作的空白画布上。
