流程搜索

使用 Process Studio 设计窗口顶部的“流程搜索”，用户可以在当前流程中轻松找到包含输入短语的环节。搜索将在流程中每个环节的名称和描述中查找短语。

搜索工具同样适用于 Object Studio 中的可视业务对象。

外观和行为

搜索栏位于靠近屏幕顶部的工具栏中。其外观如图所示：

如果找到输入的搜索短语，它所处的环节将在流程图窗口中突出显示并处于选中状态。如果找不到该短语，状态栏中将显示合适的消息。用户可以反复按 Enter 键，继续在其他环节中查找后面出现的搜索短语。

高级搜索

如果用户按下“高级搜索”选项，系统将显示“高级搜索”窗口。“高级搜索”窗口的外观如图所示：

与搜索栏一样，如果在环节的名称或描述中找到输入的搜索短语，该环节将在流程设计窗口中突出显示并处于选中状态。如果找不到该短语，状态栏中将显示一条错误消息。用户只需重新按 Enter 键或按“查找下一个”按钮，即可继续在其他环节中查找后面出现的搜索短语。

“高级搜索”窗口具有以下附加功能：

匹配大小写

如果选中了“匹配大小写”框，搜索将只查找包含具有相同大写和小写字符的相同短语的环节。例如，如果输入的搜索短语是“tEst”，将不会查找仅包含名称“test”的环节，因为大写字母和小写字母不匹配。

全字匹配

如果选中了“全字匹配”框，搜索将只查找包含与输入的搜索短语匹配的整个词汇的环节。例如，如果输入的搜索短语是“test”，则不会查找仅包含名称“test1”的环节。

向上搜索

如果选中了“向上搜索”框，此搜索功能将从当前环节向上搜索输入的搜索短语（如果没有处于活动状态的环节，则从上一个环节向上搜索），而不是向下搜索。

详细信息

“详细信息”按钮切换树视图和列表视图框的显示，如上图所示。树视图将突出显示包含输入的搜索短语的每个环节，而列表视图将突出显示最新出现的要查找的搜索短语。要更改树视图和列表视图的外观，用户可以调整“高级搜索”表单的大小，调整树视图和列表视图之间的边框，或者通过将列拖到新位置来重排列表视图中的列顺序。

正则表达式

启用正则表达式时（通过复选框），Blue Prism 会将输入到搜索框的文本解释为正则表达式。重复按 Enter 键时，每个匹配环节将依次突出显示。

请参阅通配符和正则表达式页面，了解详细信息。

元素搜索

在高级搜索中，有一个用于搜索使用某个应用程序元素的环节的复选框。例如，如果想查找对包含词汇“button”的应用程序元素的引用，您可以选中“搜索元素”复选框，所有使用包含词汇“button”的元素的环节都会被找到。

匹配大小写等熟悉的选项仍然以同样的方式提供。例如，如果使用匹配大小写选项执行上述搜索，则它将匹配元素“OK button”，但不匹配元素“Cancel Button”（由于“button”中“b”的大写和小写）。
