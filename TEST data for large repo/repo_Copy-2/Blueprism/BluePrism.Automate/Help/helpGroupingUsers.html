Grouping Users

A grouping mechanism is provided to allow users to be organized into a hierarchical tree structure for ease of management.

Groups are created and managed from the user tree control on the left of the Security – Users tab, and the context menu provides the ability to manage the groups.

Users are allocated to groups through use of drag and drop. Holding ctrl while carrying out this action will allow users who are already a member of a group to be additionally added to the selected target group.

Users can be removed from groups from the context menu, likewise groups can also be deleted in the same way.
