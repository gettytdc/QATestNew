Help - Process Grouping

This feature is subject to version restrictions. Please see the version page for further information.

Overview

Process grouping allows you to create associations between Blue Prism processes, according to the relationships you hold in your own head. The association does not change the function or behaviour of these processes - it merely provides you with the convenience of organising your processes in such a way as to make them easier to locate when browsing.

A typical use of process grouping is to group processes according to the high-level Business Processes which use them. This helps to keep track of how many processes are in use, and whether the current configuration complies with the current Blue Prism licence.

Creating Groups

Groups can be created and edited from the "Processes" section of System Manager.

Groups are created in the right-hand list, using the "New Group" button. Newly created groups are given a name by default, and this can be customised by typing a name immediately after using the "New Group" button. Groups can be renamed again at a later stage using the context menu.

Processes are added to groups by dragging and dropping them from left-to-right. A single process may belong to any number of groups. Drag and drop your processes of interest from left-to-right, dropping them onto the group(s) to which you which them to belong.

Processes may be removed from a group using the "Remove" button, or using the appropriate context menu option. This has no effect on the process — the process is not deleted, and the process is not removed from any other group than the instance selected.

Groups may be deleted using the "Delete" button, or using the appropriate context menu option. This has no effect on the processes contained in the group — the processes are not deleted.

Changes to groups are not committed until you click "Apply". This includes the creation, renaming and deletion of groups, as well as changes to group membership. To cancel any unwanted changes, simply navigate away from the screen without clicking "Apply".
