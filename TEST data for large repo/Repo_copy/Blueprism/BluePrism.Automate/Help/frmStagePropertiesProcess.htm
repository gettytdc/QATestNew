Process Stage

The process properties dialog has a number of fields to fill in.

1. Enter a name

The name that you enter identifies the process and it will be displayed on the process diagram.

2. Enter a description

The additional description field allows a lengthier narrative to be entered.

3. Choose a process

From the drop down list select a Blue Prism process to execute.

4. Edit Inputs

Edit the inputs of the process by choosing a datatype from the dropdown list. Enter an expression or drag a data item from the data explorer into the value field.

5. Edit Outputs

Edit the outputs of the process, by dragging a data item from the data explorer into the value field.
