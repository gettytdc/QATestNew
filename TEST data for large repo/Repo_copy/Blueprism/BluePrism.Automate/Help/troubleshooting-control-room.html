Troubleshooting

Control Room

Introduction

This page provides some common issues and suggested resolutions for using the Control Room feature of Blue Prism

My ResourcePC is both online and offline at the same time!

When a resource is available in control room, it is illuminated with a glowing screen; otherwise its screen is black. By hovering on a resource you may read a resource's status. An available resource may fall into one of two categories: Available and In Use.

A resource which is In Use is available for use by one user only - the same user who is logged on to that resource. Thus a user who logs on to one resource using one username and who logs on to another resource using another username may see that both resources are locally 'online' but that each resource displays the other as being offline.

There are two possible solutions to this problem:

  • Log on to each resource using the same username.
  • Make the resource available as a public resource pc.

A resource is made publicly available by running Blue Prism from the command line. The checkbox in System Manager that reads "When a user logs on to Blue Prism, start the process engine on this machine" should be unchecked first.

A resource is made exclusive to a particular user in two ways:-

  • Check the box in system manager, described above and log on to Blue Prism.
  • Run Blue Prism from the command line, under a particular username (as opposed to a public resource pc).
