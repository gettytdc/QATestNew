多组环境

安全角色权限

作为“多组环境”功能的一部分，对现有权限进行了一些更改。添加了一些权限，并且更新或重新分配了另一些权限。从 6.3 之前的版本升级后，所有现有用户仍具有其角色当前提供的访问权限。

由于“多组环境”引入的粒度提高，建议检查安全角色权限以确保所有用户仅具有必需的权限。
