ユーザーの役割許可を構成する

データゲートウェイのユーザー許可は3種類あり、適切な管理者ユーザーの役割に付与できます。

  • データゲートウェイ - 構成 - この許可を持つユーザーは、データゲートウェイ設定を構成して、ゲートウェイの出力を追加、管理できます。詳細出力またはカスタム構成の作成または編集はできません。
  • データゲートウェイ - 詳細構成 - ユーザーは、データゲートウェイ設定を構成して、ゲートウェイの出力を追加、管理できます。詳細出力およびカスタム構成の作成および編集が可能です。この許可は、出力および構成コードの編集に関する知識のある専門家ユーザーのみに付与してください。
  • データゲートウェイ - コントロールルーム - ユーザーは、コントロールルームでデータゲートウェイノードにアクセスして、データゲートウェイエンジンのステータスを確認し、必要に応じてプロセスを開始および停止します。

この許可は、アップグレードと新規インストールのためにシステム管理者に付与されます。他の組み込まれたユーザーの役割では有効になりません。
