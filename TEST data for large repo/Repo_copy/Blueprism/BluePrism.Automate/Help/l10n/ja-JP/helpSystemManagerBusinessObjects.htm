ビジネスオブジェクト

ビジネスオブジェクトビューには、利用可能なすべてのオブジェクトのリストが、そのオブジェクトグループに表示されます。

ビジネスオブジェクトの廃止

オブジェクトが使用されなくなった場合、または一時的に使用されないようにする場合は、一番上のリストから［廃止プロセス］というラベルの下のリストにオブジェクトをドラッグアンドドロップして廃止できます。廃止したオブジェクトは、他のプロセスまたはビジネスオブジェクトから呼び出すことはできません。

廃止したオブジェクトは、［オブジェクトスタジオ］メニューから編集、表示、エクスポートなどができなくなります。

この操作は、下のリストから元のリストにオブジェクトをドラッグアンドドロップすることで元に戻すことができます。この変更は即座に適用されます。
