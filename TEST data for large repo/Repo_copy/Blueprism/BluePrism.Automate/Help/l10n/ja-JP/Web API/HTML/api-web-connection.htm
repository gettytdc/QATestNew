Web接続設定

Webリクエストを行うときに適用される詳細接続設定を構成します。ユーザーの役割に対して［システム - Web接続設定］許可が有効になっているユーザーは、これらの設定を構成できます。

デフォルト設定はすべてのWeb APIリクエストに構成できます。特定のURIに値が設定されている場合、デフォルト設定は上書きされます。

  • URI - 設定が適用されるURI。URIはスキームとホストのみで構成される必要があります。設定は入力されたURIを使用するすべてのリクエストに適用されます。
  • 接続制限 - サービスポイントあたりの接続数。
  • 接続リースタイムアウト - 接続が閉じる前にアイドル状態を維持する接続に許可される時間。
  • 最大アイドル時間 - 接続が閉じられるまでの秒数。
