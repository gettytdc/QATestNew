标头

配置要作为 Web API 请求的一部分进行发送的 HTTP 标头。可以为整个 API 和单个请求方法定义标头：

  • 通用—在由 API 定义中的操作所生成的每个请求中发送的通用标头。
  • 特定于请求—仅在关联操作的请求中发送的标头。

标头是通过输入名称和所需值来定义的。值可以包括在调用 API 时以动态方式添加其值的参数。

可在通用标头和特定于请求的标头中使用相同的名称。如果是这种情况，并且两者都在 API 定义中配置了一个值，则在调用 API 时将使用特定于请求的标头的值。

  • From 标头使用在 email-address 参数中设置的值进行填充。
