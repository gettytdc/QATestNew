帮助—单点登录设置

域

您应该输入 Blue Prism 安全组所在的域。您必须输入完全限定域名（例如“global.MyDomain.local”）。

验证连接

此按钮可用于验证 Blue Prism 是否可以使用当前设备提供的名称连接到域。

管理员组

在配置 Blue Prism 的单点登录时，必须选择一个其成员将被授予访问 Blue Prism 所有区域的权限的 Active Directory 安全组。在配置 Blue Prism 单点登录时，应作为属于此安全组的用户。

为单点登录创建并配置数据库后，Blue Prism 管理员应登录 Blue Prism 并配置所需的 Blue Prism 安全角色。定义这些角色之后（每个角色都应该与 Active Directory 安全组相关联），可以使用管理角色对话框对其进行管理。

故障排除

如果遇到问题，请参阅单点登录故障排除页面。
