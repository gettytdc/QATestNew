恢复流程备份

有关 Blue Prism 自动备份功能的概述，请访问自动保存帮助页面。

1. 选择版本

对于受系统故障影响的每个流程，您都将找到原始版本和最新备份版本。选择要恢复的版本。如果您不确定，那么不必担心，因为此操作可逆（仅需再次访问恢复窗体并选择另一个版本即可）。

2. 还原

点击标记为“还原”的按钮。从此开始，下次进入 Process Studio 时，您将看到所选择的流程版本。如果您希望还原至其他版本，则您可以重新访问此窗体并还原该版本。

选择首选版本后，您应该访问 Process Studio，并点击“保存”按钮。此操作将向您提供所选版本，并清除任何现有备份。然后，您可以照常继续编辑该流程。
