Blue Prism

Largest Tables

BPDS_LargestTables

Returns Blue Prism database tables in descending, current size order (Mb). The number of tables returned can be specified as a parameter.

Source: SQL Server System Database(s).

Parameters

Parameter Name Data Type Description Validation
