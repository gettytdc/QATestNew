Edit an Existing Process

A quick and easy way to edit an existing process is by using the Edit Process wizard, from the Process Studio main menu. Follow the steps below as indicated by the numbered options on screen.

1. Locate a process

A list of existing processes from the database is presented with name, version and description details listed. The information in the box can be sorted according to each column by clicking the appropriate heading.

To select a process to edit, simply highlight the entry required using the mouse.

2. Continue to Process Studio

Once you have completed step 1 clicking OK will open the selected process and progress to Process Studio with your process loaded onto the canvas ready to start work.
