Processes

The processes view shows a list of all available business processes which have been published to the Control Room, displayed in their process groups.

To apply a process to a resource, highlight the process required and drag it over the resource you want to run it on. This will create a pending entry in the environment area, which shows all current process / resource combinations, whether they are running, stopped or pending. Newly added processes are checked in readiness for starting.

A process will show in state 'warning' if an amount of time has passed and the process is still on the same stage. This time period is configurable in the system settings, where it can also be disabled.
