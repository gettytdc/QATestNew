Data Item Watch Window

The data item watches window is used for monitoring the current value of data items during debug in Process Studio. Items are dragged from the list on the right hand side onto the left hand list. Once debug is underway, the left hand list will always display the current value of the data items it contains. When the left hand list contains a collection, the current values of each field of the current row only will be displayed as subnodes of the item. To remove items from the left hand list, drag them out of the list again.

The right hand list may be hidden from view by placing a checkmark in the checkbox labelled "Hide Data Items List". It may be shown again by clearing the checkmark.
