帮助

使用日志查看器

日志查看器提供许多用于读取和搜索会话日志的功能。

系统默认显示日志的最后一页，使用户能够从日志的最后一行确定流程退出时的情况（正常还是异常）。用户可以使用窗体底部的箭头按钮导航到其他页面：双箭头按钮将跳到最早/最新页面，单箭头按钮将导航到上一页/下一页。

要查看更大或更小的页面，请调整右下角的值，以控制每页的行数。请注意，页面越大，从数据库中获取每个页面所需的时间就越长：根据日志的大小和数据库服务器的性能，可能会需要花费一些时间。

查看环节 ID

通过窗体上半部分的复选框，可以在日志中显示或隐藏环节 ID。Blue Prism 使用环节 ID 而不是名称来标识流程图中的环节。通过显示环节 ID，高级用户可以检查流程内部的不一致性。但是，很少用户会发现此信息有用。如果不了解此功能，您可以放心地忽略它。

搜索日志

通过在窗体顶部的组合框中输入搜索词并点击“查找”按钮，用户可以搜索当前页面上的文本。搜索词的每个实例都将依次突出显示。您可以通过点击“突出显示”按钮立即查看所有实例，该按钮将使用当前颜色突出显示每个实例。您可以随后更改颜色并同时使另一个搜索词突出显示。“清除”按钮将清除所有突出显示。

系统仅在当前页面上搜索文本。但是，当到达页面末尾时，Blue Prism 将提示用户搜索下一页。此时，用户还可以选择搜索整个日志。在这种情况下，Blue Prism 将依次自动获取每个页面，搜索页面并在搜索词的下一个实例上暂停，或者在未找到实例的情况下再次获取下一页。一旦 Blue Prism 以这种方式搜索了所有页面，它将通过弹出对话框通知用户。请注意，搜索所有页面可能需要花费一些时间。

如果用户熟悉通配符和正则表达式，可以勾选相应的方框并选择适当的选项，以使用通配符和正则表达式。有关详细信息，请参阅通配符和正则表达式页面。

导出日志

如果需要在 Blue Prism 之外的上下文中使用日志，则您可以将日志导出为适合通过电子邮件发送或复制到便携式存储介质的平面文件。 有三种文件格式可供选择：纯文本文件（如日志查看器本身所示）；html 文件（适用于 Web 浏览器中的标记）；以及 csv 文件（表示逗号分隔值，此文件格式用于格式化表格数据，使其适合在电子表格中使用）。如果您选择 html 或 csv，则日志的每一行中的数据将被分为消息日期时间、消息和 stageid 等不同列（如适用）。

要导出日志（或日志的一部分），请从窗体左上角的文件菜单中选择相应的导出选项。系统将显示一个类似于弹出向导的对话框，该对话框将指导您完成必要的步骤。有关详细信息，请参阅日志导出向导的帮助文件。
