Web API

Web API 功能提供了一个接口，用于配置与提供已发布的 HTTP API 的系统和服务进行的原生交互（这些服务中最常见的是 RESTful Web 服务）。

Web API 服务功能允许 Blue Prism 流程与这些服务进行交互，以便在自动化业务流程中向这些外部系统提供数据或使用这些外部系统提供的数据或服务。Web API 功能本身提供的功能允许 Blue Prism 自动执行最常见的服务，并且可以使用代码环节扩展这些功能，以满足定制或复杂的数据结构和身份验证机制。

Web API 配置工具旨在尽可能灵活地确保复杂的 API 调用可配置。Web API 定义在 Blue Prism 的“系统”选项卡中创建，每个定义可以包含多个操作，并且每个操作通常代表 API 中的一个端点。这些操作定义了 HTTP 请求的发出方式，并确定了如何将 HTTP 响应转换为输出参数。

配置 API 定义后，可以在对象中使用关联的操作，以便流程开发人员能够将与第三方服务的交互用作自动化的一部分。
