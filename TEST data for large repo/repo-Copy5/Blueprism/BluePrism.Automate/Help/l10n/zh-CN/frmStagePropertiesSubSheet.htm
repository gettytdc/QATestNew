环节类型

页面环节

“页面引用属性”对话框中有多个字段需要填写。

1. 输入名称

您输入的名称可用于识别相应的环节，这个名称将会显示在流程示意图中。

2. 输入描述

附加的描述字段允许输入更长的叙述。

3. 选择页面

从下拉列表中，选择要执行的页面。

4. 编辑输入

编辑页面的输入项。

5. 编辑输出

编辑页面的输出项。

流程的输入参数会在此处指定。相应地，如果开始环节位于子页面上，设置的参数就与该页面有关；引用该子页面的所有环节都有可能通过页面引用环节向该页面传输值。要添加参数，只需点击“名称”列下方的网格，以便为参数添加名称。然后，在“数据类型”列下方选择数据类型。最后，通过将数据项从数据资源管理器中拖放到值列，来选择要用于存储输入参数的数据项。
