先决条件

配置数据网关之前，必须满足以下条件：

  • Blue Prism 版本必须是 6.5 企业版或更高版本。
  • 必须配置 Blue Prism 数据库所在的 SQL Server 实例，以允许数据网关引擎与之连接。这需要：
    • 在静态端口上启用 TCP/IP 协议并将其设置为侦听全部。
    • 将服务器身份验证配置为接受 SQL Server 和 Windows 身份验证两种模式。必须先在 SQL Server 中创建登录凭据，数据网关才能访问 Blue Prism 数据库。
  • Blue Prism 必须配置为使用服务器中存储的加密方案。因此，建议在运行数据网关所需的每个应用程序服务器上使用在服务器配置实用程序 (BPServer.exe) 中配置的加密方案。
