プロセスとオブジェクトのアクセス権

プロセスとオブジェクトのアクセス権はアイテムとグループの2つのタイプに分割されます。

アイテムの許可

この許可はグループ内のプロセスまたはオブジェクトに適用され、そのアイテムでユーザーができることを決定します。

許可

説明

削除

ユーザーはグループ内のアイテムを削除できます。

編集

ユーザーはグループ内のアイテムを編集できます。実行許可と表示許可も与えられます。

実行

ユーザーはアイテムを実行できますが、アイテムを開いたり、編集したりすることはできません。実行許可の場合、定義を表示する許可は自動的に与えられません。それには編集許可が必要になります。実行許可が与えられているが、定義を表示する許可や編集する許可が与えられていないユーザーは、オブジェクトまたはプロセスのデバッグ中、アイテムにステップインできません。

エクスポート

エクスポートするアクセス権が与えられていれば、ユーザーはプロセス、オブジェクト、リリースをエクスポートできます。

定義を表示する

ユーザーはオブジェクトまたはプロセスを表示できますが、実行したり、編集したりすることはできません。また、ユーザーに対して定義の表示のみが許可されているオブジェクトを参照するプロセスは、そのユーザーは正常に実行できません。そのプロセスを表示し、実行する許可がユーザーに与えられている場合は、禁止されているオブジェクトに到達したときにのみ、実行が停止されます。

Webサービスとして実行

ユーザーはWebサービスとして公開されているオブジェクトまたはプロセスを呼び出すことができます。

グループの許可

この許可によって、ユーザーがグループに関連して実行できる操作が決定されます。

許可

説明

作成

ユーザーはグループ内にアイテムを作成でき、それにはエディターで［名前を付けて保存］を使用するなどの方法があります。編集許可、実行許可、表示許可も与えられます。

グループを編集

ユーザーはスタジオツリー内で、グループを、作成、名前変更、移動、削除できます。ユーザーは、アクセス権を管理する許可が与えられていれば、プロセスを廃止したり、回収したりすることもできます。

移動に関与するあらゆる制限付きグループには、アクセス権の管理許可とグループの編集許可の両方が必要になります。詳細については、「グループを移動する」を参照してください。

インポート

ユーザーの役割でアクセス権が与えられている場合、ユーザーは［ファイル］メニューまたは［リリース］タブからアイテムをインポートできます。

アクセス権を管理

ユーザーは、グループにアクセスできる役割のアクセス権を編集することで、その役割に付与される許可を調整できます。この許可が与えられたユーザーは、ユーザーロールに設定された許可によって拒否されているアクセス権を付与することはできません。

ユーザーは自分のユーザーの役割に設定されているこの許可を更新できません。そのため、ユーザーが以前にこれらの権限を拒否されたことのあるグループに対するアクセス権管理を自分自身に付与することはできません。
