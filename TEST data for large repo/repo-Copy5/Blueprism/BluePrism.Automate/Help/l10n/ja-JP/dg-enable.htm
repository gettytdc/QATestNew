データゲートウェイプロセスを有効にする

データゲートウェイプロセスを有効にし、関連するポートをデータゲートウェイの実行に必要なすべてのアプリケーションサーバーで定義する必要があります。データゲートウェイプロセスは、単一のデバイス上の1つのインスタンスでのみ実行できます。単一のデバイスに複数のアプリケーションサーバーが構成されている場合は、それぞれに対してデータゲートウェイプロセスを有効にする必要があります。

  1. ［サーバー構成の詳細］画面にアクセスして、アプリケーションサーバーがデータゲートウェイプロセスを開始または停止するためのコマンドをリッスンするポートを入力します。

    すでにそのポートを使用しているアプリケーションサーバー上に別のプロセスがない限り、これはデフォルトポート（8101）のままにできます。

    ポート番号を変更する前に、アプリケーションサーバーを停止する必要があります。

  2. ［保存］をクリックして設定を適用します。
  3. ［サービス］メニューからBlue Prismサーバーサービスを開始します。

サービスが正常に開始されたことを確認するには、イベントビューアーを確認してください。
