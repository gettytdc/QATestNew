日別生産性

BPDS_DailyProductivity

指定期間のケースボリュームをステータス別に返します。データは完了した日に対してのみ返されます（つまり、MI統計の最終コンパイル後に発生したアクティビティは含まれません）。

ソース：Blue Prismデータベース - ライブテーブルとMIリポジトリ（有効にするにはMI統計のコンパイルが必要です）

パラメーター

パラメーター名 データ型 説明 検証
@BPQueueName テキスト 照会する個々のキューの名前（指定しない場合、すべてのキューが照会されます） なし。
@NumberOfDays 整数 レポーティング期間（日数）（指定しない場合、日数は7日になります） 最大31日
