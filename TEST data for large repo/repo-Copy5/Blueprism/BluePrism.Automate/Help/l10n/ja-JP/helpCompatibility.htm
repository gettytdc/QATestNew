製品ノート

互換性ガイド

概要

Blue Prismプラットフォームは、多数のアーキテクチャで動作し、さまざまな製品およびシステムとシームレスに連携します。ただし、特定の構成でソフトウェアを実行することに関連する互換性の問題がいくつかあります。

次に、潜在的な問題のリストを列挙します（すべての問題を網羅しているわけではありません）。

オペレーティングシステム

Windows 2000
Windows 2000では、アクティブアクセシビリティの信頼性が高くありません。特に、VB6アプリケーションとやりとりする場合に低くなります。
64ビットWindows
Blue Prismサーバーは64ビットWindowsでサポートされますが、現在、Blue Prismクライアントは64ビット環境で確実に動作できません。
Windows XP、32ビットWindows Vista、32ビットWindows 7
互換性に関する既知の問題はありません。

データベース

SQL Server 2000
SQL Server 2000では、一部のワークキューの最適化を使用できません。つまり、非常に大きなキューを使用すると、ワークキューのグラフィカルインターフェイスの動作が少し遅くなる可能性があります。
SQL Server 2005、SQL Server 2008
互換性に関する既知の問題はありません。
