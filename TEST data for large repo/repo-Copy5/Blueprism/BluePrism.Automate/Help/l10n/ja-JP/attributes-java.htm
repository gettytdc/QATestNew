アプリケーションの属性 - Javaモード

はじめに

以下は、Javaモード（Javaアプリケーション専用）のWindowsスパイツールから返される属性のリストです。 他のモードから返される属性のリストについては、以下を参照してください。

  • Win32モード
  • アクティブアクセシビリティモード
  • UIオートメーションモード
  • HTMLモード

アプリケーションの属性

アクティブ
ウィンドウが現在アクティブなウィンドウであることを示します。これには、ウィンドウ、ダイアログ、フレームなどが含まれます。 さらに、この状態は、リスト、テーブル、ツリーなど、現在アクティブなコンポーネントの子を示すために使用されます。 たとえば、アクティブなリストの子は、周囲に矩形が描かれている子です。
準備済み
要素が準備済みであることを示します。これは通常、押しているがまだ離していないボタンで使用され、マウスポインターはまだボタンの上にあります。
ビジー
現在の要素がビジーであることを示します。これは通常、移行状態にあることを示す進行状況バー、スライダー、スクロールバーなどの要素で使用されます。
チェック済み
この要素が現在選択されていることを示します。これは通常、切り替えボタン、ラジオボタン、チェックボックスなど、オン/オフ、はい/いいえ、True/Falseの状態を示す要素で使用されます。
折りたたみ済み
この要素が折りたたまれていることを示します。これは通常、拡張可能と展開済みの状態がペアになっており、段階的な表示を提供するツリーなどの要素で使用されます。
編集可能
ユーザーがこの要素のコンテンツを変更できることを示します。 これは通常、主にユーザーがテキストを入力できる要素に主に使用されます。スクロールバーやスライダーなどの他の要素は、有効になっていれば自動的に編集可能です。
有効
この要素が有効になっているかどうかを示します。ユーザーは有効になっていない要素を操作できません。そのような要素は、通常グラフィック表示でグレー表示されます。
拡張可能
この要素は子を段階的に表示できることを示します。これは通常、ツリーなどの階層要素で使用され、多くの場合展開済みまたは折りたたみ済みの状態がペアになっています。
展開済み
この要素が展開されていることを示します。これは通常、拡張可能と折りたたみ済みの状態がペアになっており、段階的に表示するツリーなどの要素で使用されます。
フォーカス可能
この要素がキーボードフォーカスを受け入れることができることを示します。フォーカス可能であるということは、キーボードの入力で生じるすべてのイベントは、通常、それにフォーカスがあるときに渡されるということです。 フォーカスも参照してください。
フォーカス
この要素には現在キーボードフォーカスがあることを示します。フォーカスされた要素はキーボード入力を受け取る要素です。 フォーカス可能も参照してください。
水平
この要素の向きが垂直ではなく水平であることを示します。これは、スクロールバーやトラックバーなど、複数の向きで表示できる要素に関係します。
アイコン化
この要素は最小化され、アイコンでのみ表されることを示します。これは通常、フレームと内部フレームにのみ関連付けられています。
モーダル
ユーザーが別のウィンドウで要素を操作する前に、この要素で何かを実行する必要があることを示します。 これは通常、ダイアログにのみ関連付けられています。
複数行
この（テキスト）要素に複数行のテキストを含めることができることを示します。単一行も参照してください。
複数選択可能
この要素が、複数の子を同時に選択済みにできることを示します。
不透明
この要素が四角形領域内のすべてのピクセルをペイントすることを示します。不透明でないコンポーネントは、そのピクセルの一部のみをペイントし、その下にあるピクセルが「透けて見える」ようにします。そのため、ピクセルを完全にペイントしないコンポーネントは、ある程度の透明度があります。
押された状態
この要素が現在押されていることを示します。これは通常ボタンに関連付けられており、ユーザーは、ボタン上にポインターがある間にマウスボタンを押し、まだマウスボタンを離していないことを示します。
サイズ変更可能
この要素のサイズが固定されていないことを示します。
選択可能
この要素は、他の子を選択済みにできる要素の子であり、この子は、ユーザーの希望に応じて選択できる子の1つであることを示します。
選択済み
この要素は、他の子を選択済みにできる要素の子であり、この子は、選択されている子の1つであることを示します。
表示中
この要素、要素の親、要素の親の親などがすべて表示中であることを示します。なお、これは必ずしも要素が画面上に表示されていることを示すわけではありません。他の表示中の要素の背後に隠れていることもあります。
単一行
この（テキスト）要素に単一行のテキストのみを含めることができることを示します。これは複数行要素とは対照的です。
一時的
この要素が一時的であることを示します。つまり、ユーザーインターフェイスで他の要素を支援することを除けば、この要素はユーザーインターフェイスで直接的な役割を果たしません。一時的な要素は通常、自動化の目的には役立たず、待機クエリを節約します。
垂直
この要素の方向が水平ではなく垂直であることを示します。これは通常、スクロールバーやトラックバーなど、複数の向きで表示できる要素に関連付けられています。
可視
この要素が表示されていることを示します。注意：これは要素が表示されることを意図していますが、含まれている要素の1つが表示されないため、実際には画面に表示されていない可能性があります。
