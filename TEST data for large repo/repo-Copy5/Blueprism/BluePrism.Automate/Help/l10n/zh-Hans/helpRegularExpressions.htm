正则表达式

简介

Blue Prism 使用 .NET framework 的正则表达式引擎。正则表达式模式由特殊的语法或语言定义，与 Perl 5 正则表达式兼容。

正则表达式引擎使用默认选项设置调用。

特殊字符

以下字符是正则表达式语法的一部分。要在正则表达式中按原样使用以下任何字符，必须在这些字符前面加上转义符。

\转义符。
[字符组开始。
]字符组结束。
.通配符，匹配除换行符以外的任何单个字符。
^匹配必须发生在字符串的开头。
$匹配必须发生在字符串的末尾或换行符之前。
(组开始。
)组结束。
*匹配上一个元素零次或多次。
+匹配上一个元素一次或多次。
?匹配上一个元素零次或一次。
{限定符开始
}限定符结束
|匹配由 | 字符分隔的任何一个元素。
