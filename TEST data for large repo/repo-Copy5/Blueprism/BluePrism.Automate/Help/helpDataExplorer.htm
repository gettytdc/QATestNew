Data Explorer Tree

Many of the stage properties dialogs have a tree like data explorer panel on the right hand side.

This tree view gives an alphabetically sorted list of all the data items on the current process.

Group by Sheet

To make finding data items easier you can group by sheet, this means that there will be a sub branch on the tree with the name of each of the processes sheets and all the data items of these sheets will be categorised under these branches.

Group by Data Type

The data items can also be grouped by data type, this means that there will be a sub branch on the tree with the name of each data type.
