Stage Types

Write Stage

The write stage allows data to be written to your target application from data items.

Properties Form

The properties form works analogously to the reader stage properties form, except that the flow of data is in the opposite direction (ie from the business object to the application instead of from the application to the business object). For this reason, the features of the form have been reversed: the application elements are filled in on the right and the data items are filled in on the left.

Notwithstanding these visual differences, the overriding behaviour is the same. Please visit the reader stage help for details of how to use the form.
