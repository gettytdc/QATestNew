Versions

Newer versions of Blue Prism have more features, if you are interested in a feature that does not exist in your version of Blue Prism please contact us.

Below is a breakdown of key features that have been added to each version of Blue Prism

Version 5.0.x

  • Insight and analysis
    • Management dashboards
    • Trend analysis
    • Utilization and completion rates
    • Dependency tracking
  • Digital Workforce
    • Active queues
    • Control room filters
    • Additional logging levels
  • Scalability
    • Multi-tier grouping structures
    • Credential manager expansion
    • Better support for higher latency environments
    • Support for locales and international character sets
  • Development and implementation
    • Global application models
    • Copy and paste actions or pages
    • Font recognition for joined characters
    • Release manager updates
    • OCR functionality
    • Version history prominence (Processes and Objects)
  • User Experience
    • Refreshed user interface
    • Combined Process and Object Studio
    • Environment themes
  • Enterprise Platform
    • .NET 4 platform support
    • SQL Azure support
    • SQL AlwaysOn Availability Groups support
    • Published SQL roles for controlling database securables

Version 4.2.x

  • 64 Bit Support
  • Better User Access Control Support
  • Improved Character Matching / Region Editor
  • Calendars Internal Business Object
  • Application Model filtering
  • Improved Session Log searching/filtering
  • Encrypted Work Queues
  • Encryption Business Object
  • Scheduled tasks continuing after failed sessions
  • Improved Code Stage editor
  • Multiline text support in text data items
  • New Datatype conversion functions
  • More powerful password policy rules

Version 4.1.x

  • Release Manager
  • Design Control - Advanced Validation
  • Font Recognition
  • Memory Diagnostics
  • Environment Constants
  • Session Variables - Editable from Control Room
  • Utility VBOs included in product

Version 4.0.x

  • Nested Collections
  • Compound Objects (implemented as single row collections)
  • Enhanced Web Service support

Version 3.5.x

  • Work Queues Management
  • Exception Handling
  • Credentials Manager
  • Resource Pools
  • Process Grouping
  • Scheduler
  • Blue Prism Server

Version 3.0.x

  • Object Studio
  • Application Modeller (also known as the Integration Assistant)
  • Process MI

Version 2.1.x

  • Log File Export
  • Debug Improvements (variable speed, step out)
  • Create Collection
  • Publish / Retire Processes
  • Process Differ
  • Data Item Watches

Version 2.0.x

  • Process Audit
  • Process Backup
  • Process Search
  • Improved Stage Properties GUI
  • Control Room performance enhancements

Version 1.1.x

  • Business Object Configuration
  • Pages in Process Studio
